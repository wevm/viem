# `SoladySmartAccount.from`

Creates a Solady Smart Account. EntryPoint 0.7 and its compatible factory are used by default. Supplying an EntryPoint requires a matching factory deployment.

:::info
`signMessage` and `signTypedData` use ERC-7739 nested signing. Counterfactual account signatures are wrapped with ERC-6492. The default EntryPoint 0.7 factory supports ERC-7739; custom factories must deploy a compatible implementation.
:::

## Usage

```ts
import { Account } from 'viem'
import { SoladySmartAccount } from 'viem/account-abstraction'

const owner = Account.fromPrivateKey('0x…')

const account = await SoladySmartAccount.from({
  client: executionClient,
  owner,
})
```

### EntryPoint 0.6

```ts
import { EntryPoint, SoladySmartAccount } from 'viem/account-abstraction'

const account = await SoladySmartAccount.from({
  client: executionClient,
  entryPoint: {
    abi: EntryPoint.abiV06,
    address: EntryPoint.addressV06,
    version: '0.6',
  },
  factoryAddress: '0x…',
  owner,
})
```

`factoryAddress` must deploy a Solady account implementation whose `entryPoint()` targets the selected EntryPoint. Solady's canonical implementation and the default factory target EntryPoint 0.7.

## Parameters

### options

#### options.address (optional)

- **Type:** `Address`

Existing account address. When omitted, it is derived from the factory.

#### options.client

- **Type:** `Client`

Execution Client used to read account and EntryPoint state.

#### options.entryPoint (optional)

- **Type:** `{ abi: Abi, address: Address, version: '0.6' | '0.7' }`
- **Default:** EntryPoint 0.7

EntryPoint supported by the account deployment.

#### options.factoryAddress (optional)

- **Type:** `Address`
- **Default:** Solady EntryPoint 0.7 factory

Solady Smart Account factory address. Required whenever `options.entryPoint` is supplied.

#### options.getNonce (optional)

- **Type:** `SmartAccount.Implementation['getNonce']`

Overrides the account nonce resolver.

#### options.owner

- **Type:** `Address | Account`

Account owner. Address owners sign through the execution Client; local owners sign locally.

#### options.salt (optional)

- **Type:** `Hex`
- **Default:** `'0x0'`

Account deployment salt.

## Return Type

`SoladySmartAccount.Account`
