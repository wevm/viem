# `WebAuthnAccount.from`

Creates an owner account from a WebAuthn P256 credential. The result can own a [`CoinbaseSmartAccount`](/account-abstraction/accounts/coinbase).

## Create a Credential

Create credentials with the `WebAuthnP256` utilities, and serialize them for storage with `Credential`.

```ts
import { WebAuthnP256 } from 'viem/utils'
import { Credential } from 'viem/account-abstraction'
import { WebAuthnAccount } from 'viem/account-abstraction'

const credential = await WebAuthnP256.createCredential({ name: 'Example' })
const account = WebAuthnAccount.from(credential)
```

Use `Credential.serialize` before JSON persistence because the live credential contains bigint and `ArrayBuffer` values. Deserialize it when loading:

```ts
const serialized = Credential.serialize(credential)
const json = JSON.stringify(serialized)

const restored = Credential.deserialize(JSON.parse(json))
const account = WebAuthnAccount.from(restored)
```

## Parameters

### credential

- **Type:** `{ id: string, publicKey: P256PublicKey | Hex }`

WebAuthn credential or its persisted identity.

### options (optional)

#### options.getFn (optional)

- **Type:** `WebAuthnP256.sign.Options['getFn']`
- **Default:** `window.navigator.credentials.get`

Credential request function for custom runtimes and tests.

#### options.rpId (optional)

- **Type:** `string`

Relying Party ID supplied during authentication.

## Return Type

`WebAuthnAccount.Account`

The account exposes `id`, `publicKey`, `sign`, `signMessage`, `signTypedData`, and `type: 'webAuthn'`.
