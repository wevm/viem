# `userOperation.send`

Broadcasts a User Operation to the Bundler.

## Usage

:::code-group

```ts twoslash [example.ts]
import { Value } from 'viem/utils'
import { account, bundlerClient } from './config'

const hash = await bundlerClient.userOperation.send({ // [!code focus:7]
  account,
  calls: [{
    to: '0x70997970c51812dc3a010c7d01b50e0d17dc79c8',
    value: Value.fromEther('1')
  }],
})
```

```ts twoslash [config.ts] filename="config.ts"
import { Client, http } from 'viem'
import { BundlerClient, Simple7702SmartAccount } from 'viem/erc4337'
import { Account } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http()
})

export const account = await Simple7702SmartAccount.from({
  client,
  owner: Account.fromPrivateKey('0x...'),
})

export const bundlerClient = BundlerClient.create({
  client,
  transport: http('https://public.pimlico.io/v2/1/rpc')
})
```

:::

:::info
The Bundler URL above is a public endpoint. Please do not use it in production as you will likely be rate-limited. Consider using [Pimlico's Bundler](https://www.pimlico.io), [Biconomy's Bundler](https://www.biconomy.io), or another Bundler service.
:::

Pass either `calls` for Smart Account encoding or an already encoded `callData`. A complete User Operation can be sent without an account when `entryPointAddress`, `sender`, and `signature` are provided.

### Simple7702 Authorization

The first User Operation from a [`Simple7702SmartAccount`](/account-abstraction/accounts/simple-7702) needs a signed EIP-7702 authorization.

```ts
import { Actions as CoreActions } from 'viem'

const authorization = await CoreActions.wallet.signAuthorization(
  client,
  account.authorization,
)

const hash = await bundlerClient.userOperation.send({
  authorization,
  calls: [{ to: '0x0000000000000000000000000000000000000000' }],
})
```

Later User Operations can omit `authorization` after the account is delegated.

### Account Hoisting

If you do not wish to pass an `account` to every `userOperation.send`, you can also hoist the Account on the Bundler Client (see `config.ts`).

[Learn more](/account-abstraction/clients/bundler#account-optional).

:::code-group

```ts twoslash [example.ts]
import { Value } from 'viem/utils'
import { bundlerClient } from './config'

const hash = await bundlerClient.userOperation.send({ // [!code focus:7]
  calls: [{
    to: '0x70997970c51812dc3a010c7d01b50e0d17dc79c8',
    value: Value.fromEther('1')
  }],
})
```

```ts twoslash [config.ts] filename="config.ts"

import { BundlerClient, Simple7702SmartAccount } from 'viem/erc4337'

```

### Contract Calls

The `calls` property also accepts **Contract Calls**, and can be used via the `abi`, `functionName`, and `args` properties.

:::code-group

```ts twoslash [example.ts]
import { bundlerClient } from './config'
import { wagmiAbi } from './abi' // [!code focus]

const hash = await bundlerClient.userOperation.send({ // [!code focus:7]
  calls: [{
    abi: wagmiAbi,
    functionName: 'mint',
    to: '0xFBA3912Ca04dd458c843e2EE08967fC04f3579c2',
  }],
})
```

```ts twoslash [abi.ts] filename="abi.ts"
export const wagmiAbi = [
  // ...
  {
    inputs: [],
    name: "mint",
    outputs: [],
    stateMutability: "nonpayable",
    type: "function",
  },
  // ...
] as const;
```

```ts twoslash [config.ts]
import { Client, http } from 'viem'
import { BundlerClient, Simple7702SmartAccount } from 'viem/erc4337'
import { Account } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http()
})

  client,

  account,
  client,

})
```

:::

## Returns

`Hash`

The User Operation hash.

## Parameters

### account (Optional)

- **Type:** `SmartAccount`
- **Default:** `bundlerClient.account`

The Account to use for User Operation execution.

```ts twoslash
import { Value } from 'viem/utils'
import { account, bundlerClient } from './config'
const hash = await bundlerClient.userOperation.send({
  account, // [!code focus]
  calls: [{
    to: '0x70997970c51812dc3a010c7d01b50e0d17dc79c8',
    value: Value.fromEther('1')
  }]
})
```

### calls (Optional)

- **Type:** `({ data?: Hex | undefined, to: Address, value?: bigint | undefined } | { abi: Abi, functionName: string, args: unknown[], to: Address, value?: bigint | undefined })[]`

The calls to execute in the User Operation.

```ts twoslash
import { account, bundlerClient } from './config'
import { Value } from 'viem/utils'
import { wagmiAbi } from './abi'
const hash = await bundlerClient.userOperation.send({
  account,
  calls: [{ // [!code focus]
    to: '0x70997970c51812dc3a010c7d01b50e0d17dc79c8', // [!code focus]
    value: Value.fromEther('1') // [!code focus]
  }, { // [!code focus]
    abi: wagmiAbi, // [!code focus]
    functionName: 'mint', // [!code focus]
    to: '0xFBA3912Ca04dd458c843e2EE08967fC04f3579c2', // [!code focus]
  }] // [!code focus]
})
```

:::tip
You can also pass raw call data via the `callData` property:

```ts twoslash
import { account, bundlerClient } from './config'
const hash = await bundlerClient.userOperation.send({
  account,
  callData: '0xdeadbeef', // [!code focus]
})
```
:::

### authorization (Optional)

- **Type:** `Authorization.Signed`

Signed EIP-7702 authorization for the first User Operation from a delegated account.

### callGasLimit (Optional)

- **Type:** `bigint`

The amount of gas to allocate the main execution call.


### dataSuffix (Optional)

- **Type:** `Hex`

Data to append to the end of User Operation calldata. Useful for adding [transaction attribution](https://oxlib.sh/ercs/erc8021/Attribution).

If not provided, the `dataSuffix` configured on the Bundler Client (or inherited from the underlying client) will be used.

```ts twoslash
import { account, bundlerClient } from './config'
import { Value } from 'viem/utils'
const hash = await bundlerClient.userOperation.send({
  account,
  calls: [{
    to: '0x70997970c51812dc3a010c7d01b50e0d17dc79c8',
    value: Value.fromEther('1')
  }],
  dataSuffix: '0xdeadbeef', // [!code focus]
})
```

### entryPointAddress (Optional)

- **Type:** `Address`
- **Default:** `account.entryPoint.address`

EntryPoint that receives the User Operation. This is required when sending a complete User Operation without a Smart Account.

### paymaster (Optional)

- **Type:** `Address | true | PaymasterClient | PaymasterActions`

Sets Paymaster configuration for the User Operation.

- If `paymaster: Address`, it will use the provided Paymaster contract address for sponsorship.
- If `paymaster: PaymasterClient`, it will use the provided [Paymaster Client](/account-abstraction/clients/paymaster) for sponsorship.
- If `paymaster: true`, it will be assumed that the Bundler Client also supports Paymaster RPC methods (e.g. `pm_getPaymasterData`), and use them for sponsorship.
- If [custom functions](/account-abstraction/clients/bundler#paymastergetdata-optional) are provided to `paymaster`, it will use them for sponsorship.

#### Using a Paymaster Contract Address

```ts twoslash
import { account, bundlerClient } from './config'
import { Value } from 'viem/utils'
const hash = await bundlerClient.userOperation.send({
  account,
  calls: [{
    to: '0x70997970c51812dc3a010c7d01b50e0d17dc79c8',
    value: Value.fromEther('1')
  }],
  paymaster: '0x942fD5017c0F60575930D8574Eaca13BEcD6e1bB', // [!code focus]
  paymasterData: '0xdeadbeef',
})
```

#### Using a Paymaster Client

```ts twoslash
import { account, bundlerClient } from './config'
import { http } from 'viem'
import { Value } from 'viem/utils'
import { PaymasterClient } from 'viem/erc4337'
const paymasterClient = PaymasterClient.create({ // [!code focus]
  transport: http('https://api.pimlico.io/v2/1/rpc?apikey={API_KEY}') // [!code focus]
}) // [!code focus]

const hash = await bundlerClient.userOperation.send({
  account,
  calls: [{
    to: '0x70997970c51812dc3a010c7d01b50e0d17dc79c8',
    value: Value.fromEther('1')
  }],
  paymaster: paymasterClient, // [!code focus]
})
```

#### Using the Bundler Client as Paymaster

```ts twoslash
import { account, bundlerClient } from './config'
import { Value } from 'viem/utils'
const hash = await bundlerClient.userOperation.send({
  account,
  calls: [{
    to: '0x70997970c51812dc3a010c7d01b50e0d17dc79c8',
    value: Value.fromEther('1')
  }],
  paymaster: true, // [!code focus]
})
```
