# `l1.getL2Output`

Retrieves the L2 output covering an L2 block number or super-root timestamp.

## Usage

Read the first legacy output or respected dispute-game claim after an L2 position.

:::code-group

```ts twoslash [example.ts]
import { optimism } from 'viem/chains'
import { client } from './viem.config'

const output = await client.output.getL2Output({ // [!code focus]
  l2BlockNumber: 123_456_789n, // [!code focus]
  targetChain: optimism, // [!code focus]
}) // [!code focus]
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
// [!include ~/snippets/op-stack/l1.config.ts:setup]
```

:::

### Standalone Action

Call `Actions.l1.getL2Output` directly by passing the [Client](/docs/clients) as the first argument.

:::code-group

```ts twoslash [example.ts]
import { optimism } from 'viem/chains'
import { Actions } from 'viem/op-stack'
import { client } from './viem.config'

const output = await Actions.l1.getL2Output(client, { // [!code focus]
  l2BlockNumber: 123_456_789n, // [!code focus]
  targetChain: optimism, // [!code focus]
}) // [!code focus]
```

## Return Type

```ts
{
  l2BlockNumber: bigint
  outputIndex: bigint
  outputRoot: Hex
  timestamp: bigint
}
```

The legacy output proposal or dispute-game claim that covers the requested L2 position.

## Parameters

### l2BlockNumber

- **Type:** `bigint`

The L2 block number, or timestamp for super-root games.

### limit (Optional)

- **Type:** `number`
- **Default:** `100`

The maximum number of recent dispute games to inspect.

### targetChain (Optional)

- **Type:** `Chain`

The OP Stack L2 [Chain](/docs/chains) whose L1 contracts are used.

### disputeGameFactoryAddress (Optional)

- **Type:** `Address`

An explicit Dispute Game Factory address. Supply all three explicit addresses when omitting `targetChain`.

### l2OutputOracleAddress (Optional)

- **Type:** `Address`

An explicit L2 Output Oracle address. Supply all three explicit addresses when omitting `targetChain`.

### portalAddress (Optional)

- **Type:** `Address`

An explicit Optimism Portal address. Supply all three explicit addresses when omitting `targetChain`.

### chain (Optional)

- **Type:** `Chain`
- **Default:** `client.chain`

The L1 chain used to read the contracts.
