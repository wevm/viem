# `l1.estimateFinalizeWithdrawalGas`

Estimates the gas required to finalize an OP Stack withdrawal on L1.

## Usage

Estimate finalization gas for a withdrawal built from its L2 receipt.

:::code-group

```ts twoslash [example.ts]
import { optimism } from 'viem/chains'
import type { Withdrawal } from 'viem/op-stack'
import { client } from './viem.config'

declare const withdrawal: Withdrawal.Withdrawal

const gas = await client.withdrawal.estimateFinalizeWithdrawalGas({ // [!code focus]
  targetChain: optimism, // [!code focus]
  withdrawal, // [!code focus]
}) // [!code focus]
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
// [!include ~/snippets/op-stack/l1.config.ts:setup]
```

:::

### Standalone Action

Call `Actions.l1.estimateFinalizeWithdrawalGas` directly by passing the [Client](/docs/clients) as the first argument.

:::code-group

```ts twoslash [example.ts]
import { optimism } from 'viem/chains'
import { Actions, type Withdrawal } from 'viem/op-stack'
import { client } from './viem.config'

declare const withdrawal: Withdrawal.Withdrawal

const gas = await Actions.l1.estimateFinalizeWithdrawalGas(client, { // [!code focus]
  targetChain: optimism, // [!code focus]
  withdrawal, // [!code focus]
}) // [!code focus]
```

## Return Type

`bigint`

The estimated L1 gas.

## Parameters

### withdrawal

- **Type:** `Withdrawal.Withdrawal`

The withdrawal to finalize.

### proofSubmitter (Optional)

- **Type:** `Address | null`

The proof submitter used by portals that support external-proof finalization.

### account (Optional)

- **Type:** `Account | Address`

The [Account](/docs/accounts) used for gas estimation.

### targetChain (Optional)

- **Type:** `Chain`

The OP Stack L2 [Chain](/docs/chains) whose portal contract is used.

### portalAddress (Optional)

- **Type:** `Address`

An explicit Optimism Portal address. When supplied, `targetChain` is optional.

### chain (Optional)

- **Type:** `Chain`
- **Default:** `client.chain`

The L1 chain used for gas estimation.

### gas (Optional)

- **Type:** `bigint`

The gas limit included in the estimate request.

### maxFeePerGas (Optional)

- **Type:** `bigint`

The maximum total fee per gas, in wei.

### maxPriorityFeePerGas (Optional)

- **Type:** `bigint`

The maximum priority fee per gas, in wei.

### nonce (Optional)

- **Type:** `number`

The transaction nonce included in the estimate request.
