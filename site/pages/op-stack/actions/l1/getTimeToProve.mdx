# `l1.getTimeToProve`

Returns the remaining time before an OP Stack withdrawal can be proved.

## Usage

Estimate when the withdrawal receipt will be covered by a legacy output or respected dispute game.

:::code-group

```ts twoslash [example.ts]
import { optimism } from 'viem/chains'
import type { TransactionReceipt } from 'viem/op-stack'
import { client } from './viem.config'

declare const receipt: TransactionReceipt.TransactionReceipt

const time = await client.withdrawal.getTimeToProve({ // [!code focus]
  receipt, // [!code focus]
  targetChain: optimism, // [!code focus]
}) // [!code focus]
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
// [!include ~/snippets/op-stack/l1.config.ts:setup]
```

:::

### Standalone Action

Call `Actions.l1.getTimeToProve` directly by passing the [Client](/docs/clients) as the first argument.

:::code-group

```ts twoslash [example.ts]
import { optimism } from 'viem/chains'
import { Actions, type TransactionReceipt } from 'viem/op-stack'
import { client } from './viem.config'

declare const receipt: TransactionReceipt.TransactionReceipt

const time = await Actions.l1.getTimeToProve(client, { // [!code focus]
  receipt, // [!code focus]
  targetChain: optimism, // [!code focus]
}) // [!code focus]
```

## Return Type

```ts
{
  interval: number
  seconds: number
  timestamp?: number
}
```

The estimated interval, remaining seconds, and optional submission timestamp in milliseconds.

## Parameters

### receipt

- **Type:** `TransactionReceipt`

The L2 withdrawal transaction receipt.

### targetChain (Optional)

- **Type:** `Chain`

The OP Stack L2 [Chain](/docs/chains) whose L1 contracts are used.

### disputeGameFactoryAddress (Optional)

- **Type:** `Address`

An explicit Dispute Game Factory address. Supply all three explicit addresses when omitting `targetChain`.

### l2OutputOracleAddress (Optional)

- **Type:** `Address`

An explicit L2 Output Oracle address. Supply all three explicit addresses when omitting `targetChain`.

### portalAddress (Optional)

- **Type:** `Address`

An explicit Optimism Portal address. Supply all three explicit addresses when omitting `targetChain`.

### chain (Optional)

- **Type:** `Chain`
- **Default:** `client.chain`

The L1 chain used to read the contracts.

### intervalBuffer (Optional)

- **Type:** `number`
- **Default:** `1.1`

The multiplier applied to the observed output or game interval.

### l2Timestamp (Optional)

- **Type:** `bigint`

The L2 timestamp used as the position for super-root games. Defaults to `receipt.blockNumber` for other portal versions.
