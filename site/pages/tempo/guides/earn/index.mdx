# Earn

## Overview

Earn vaults put deposited assets into a yield venue and issue EarnShare that represent the
position. The asset value of those shares can change as the venue earns yield and the vault accrues
fees.

Each action takes a EarnVault address. Use
[`earn.getEarnVault`](/tempo/actions/earn.getEarnVault) to inspect its asset token, current venue integration,
state, and supported actions before depositing.

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

const earnVault = await client.earn.getEarnVault({
  earnVault: '0x0000000000000000000000000000000000000001',
})
// @log: {
// @log:   assetToken: '0x20c0...0001',
// @log:   capabilities: { exactWithdraw: true, inKindDeposit: true, redeem: true, ... },
// @log:   depositsPaused: false,
// @log:   earnShare: '0x20c0...0002',
// @log:   ...
// @log: }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
// [!include ~/snippets/tempo/viem.config.ts:setup]
```

:::

Assets are the TIP-20 tokens accepted by a earnVault. EarnShare are TIP-20 tokens representing a
proportional claim on the vault's assets. Venue shares are positions held by the vault in its
underlying yield venue.
