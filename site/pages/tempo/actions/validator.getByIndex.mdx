# validator.getByIndex

Gets a validator address by index.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const validatorAddress = await client.validator.getByIndex({
  index: 0n,
})
// @log: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8'
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const validatorAddress = await Actions.validator.getByIndex(client, {
  index: 0n,
})
// @log: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8'
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Parameters

### index

- **Type:** `bigint`

Validator index.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const validatorAddress = await Actions.validator.getByIndex(client, {
  index: 1n, // [!code focus]
})
```

## Return Value

`Address`

The validator address at the given index.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The ValidatorConfig read failed. |
