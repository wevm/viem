# nonce.getNonce

Gets the nonce for an account and nonce key.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const value = await client.nonce.getNonce({
  account: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  nonceKey: 1n,
})
// @log: 0n
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const value = await Actions.nonce.getNonce(client, {
  account: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  nonceKey: 1n,
})
// @log: 0n
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Parameters

### account

- **Type:** `Address`

Account address.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const value = await Actions.nonce.getNonce(client, {
  account: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
  nonceKey: 1n,
})
```

### nonceKey

- **Type:** `bigint`

Nonce key (must be greater than 0, key 0 is reserved for protocol nonces).

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const value = await Actions.nonce.getNonce(client, {
  account: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  nonceKey: 42n, // [!code focus]
})
```

## Return Value

`bigint`

The nonce for the account and key.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The NonceManager read failed. |
