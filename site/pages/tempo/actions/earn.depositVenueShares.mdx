# `earn.depositVenueShares`

Deposits venue shares into a vault and mints EarnShare to a recipient.

## Usage

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

const result = await client.earn.depositVenueSharesSync({
  earnShares: 499_000_000n,
  slippageBps: 30,
  earnVault: '0x0000000000000000000000000000000000000002',
  venueShares: 500_000_000n,
  venueShareToken: '0x20c0000000000000000000000000000000000001',
})
// @log: {
//   caller: '0x...',
//   earnShares: 498_750_000n,
//   receipt: { ... },
//   receivedVenueShares: 500_000_000n,
//   recipient: '0x...',
//   venueShares: 500_000_000n,
// }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
// [!include ~/snippets/tempo/viem.config.ts:setup]
```

:::

The action reads the current venue integration, approves it to pull `venueShareToken`, and deposits
the venue shares in one transaction.

### Asynchronous Usage

`earn.depositVenueSharesSync` waits for inclusion and returns the decoded deposit event. Use
`earn.depositVenueShares` to return the transaction hash immediately.

```ts twoslash
import { client } from './viem.config'

const hash = await client.earn.depositVenueShares({
  minEarnShares: 497_000_000n,
  earnVault: '0x0000000000000000000000000000000000000002',
  venueShares: 500_000_000n,
  venueShareToken: '0x20c0000000000000000000000000000000000001',
})
// @log: 0x1234...abcd
```

## Return Value

```ts
type ReturnValue = {
  /** Depositing caller. */
  caller: Address
  /** EarnShare minted. */
  earnShares: bigint
  /** Transaction receipt. */
  receipt: TransactionReceipt
  /** Venue shares measured as received by the integration. */
  receivedVenueShares: bigint
  /** EarnShare recipient. */
  recipient: Address
  /** Venue shares requested for transfer. */
  venueShares: bigint
}
```

The asynchronous action returns the transaction hash instead.

## Parameters

### earnShares (optional)

- **Type:** `bigint`

Quoted EarnShare output. Provide it with `slippageBps`, or provide `minEarnShares` instead.

### minEarnShares (optional)

- **Type:** `bigint`

Minimum EarnShare to accept. It must be greater than zero and cannot be combined with
`earnShares` or `slippageBps`.

### recipient (optional)

- **Type:** `Address`
- **Default:** `account.address`

EarnShare recipient.

### slippageBps (optional)

- **Type:** `number`

Slippage tolerance below `earnShares`, in basis points. For example, `30` means 0.3%.

### earnVault

- **Type:** `Address`

EarnVault address.

### venueShares

- **Type:** `bigint`

Venue shares to deposit, in base units.

### venueShareToken

- **Type:** `Address`

Venue share token to deposit.
