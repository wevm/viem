# policy.setAdminSync

Sets the admin for a policy, and waits for the transaction to be confirmed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { admin, updater } = await client.policy.setAdminSync({
  admin: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  policyId: 2n,
})
// @log: { policyId: 2n, updater: '0x…', admin: '0x…', receipt: { … } }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const { admin, updater } = await Actions.policy.setAdminSync(client, {
  admin: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  policyId: 2n,
})
// @log: { policyId: 2n, updater: '0x…', admin: '0x…', receipt: { … } }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### admin

- **Type:** `Address`

New admin address.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.policy.setAdminSync(client, {
  admin: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8', // [!code focus]
  policyId: 2n,
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### policyId

- **Type:** `bigint`

Policy ID.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.policy.setAdminSync(client, {
  admin: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  policyId: 2n, // [!code focus]
})
```

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

## Return Value

`{ policyId: bigint, updater: Address, admin: Address, receipt: TransactionReceipt }`

The `PolicyAdminUpdated` event data and the transaction receipt.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed or the receipt reverted. |
