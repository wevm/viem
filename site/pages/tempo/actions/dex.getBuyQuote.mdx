# dex.getBuyQuote

Gets the quote for buying a specific amount of tokens. Quotes are read-only and do not execute a swap.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const amountIn = await client.dex.getBuyQuote({
  amountOut: 100n,
  tokenIn: '0x20c0000000000000000000000000000000000001',
  tokenOut: '0x20c0000000000000000000000000000000000002',
})
// @log: 101n
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const amountIn = await Actions.dex.getBuyQuote(client, {
  amountOut: 100n,
  tokenIn: '0x20c0000000000000000000000000000000000001',
  tokenOut: '0x20c0000000000000000000000000000000000002',
})
// @log: 101n
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Parameters

### amountOut

- **Type:** `bigint`

Amount of `tokenOut` to buy.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const amountIn = await Actions.dex.getBuyQuote(client, {
  amountOut: 100n, // [!code focus]
  tokenIn: '0x20c0000000000000000000000000000000000001',
  tokenOut: '0x20c0000000000000000000000000000000000002',
})
```

### tokenIn

- **Type:** `TokenId | Address`

Address or ID of the token to spend.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const amountIn = await Actions.dex.getBuyQuote(client, {
  amountOut: 100n,
  tokenIn: 1n, // [!code focus]
  tokenOut: '0x20c0000000000000000000000000000000000002',
})
```

### tokenOut

- **Type:** `TokenId | Address`

Address or ID of the token to buy.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const amountIn = await Actions.dex.getBuyQuote(client, {
  amountOut: 100n,
  tokenIn: '0x20c0000000000000000000000000000000000001',
  tokenOut: 2n, // [!code focus]
})
```

## Return Value

`bigint`

The amount of `tokenIn` needed to buy the specified `amountOut`.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The quote read failed. |
