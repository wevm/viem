# validator.list

Gets the complete set of validators.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const validators = await client.validator.list()
// @log: [
// @log:   {
// @log:     publicKey: '0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef',
// @log:     active: true,
// @log:     index: 0n,
// @log:     validatorAddress: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
// @log:     inboundAddress: '192.168.1.1:8080',
// @log:     outboundAddress: '192.168.1.1:8080',
// @log:   },
// @log: ]
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const validators = await Actions.validator.list(client)
// @log: [
// @log:   {
// @log:     publicKey: '0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef',
// @log:     active: true,
// @log:     index: 0n,
// @log:     validatorAddress: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
// @log:     inboundAddress: '192.168.1.1:8080',
// @log:     outboundAddress: '192.168.1.1:8080',
// @log:   },
// @log: ]
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Return Value

`readonly { publicKey: Hex, active: boolean, index: bigint, validatorAddress: Address, inboundAddress: string, outboundAddress: string }[]`

Array of all validators with their information.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The ValidatorConfig read failed. |
