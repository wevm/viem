# `token.getTotalSupply`

Gets the total supply of a TIP-20 token.

## Usage

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

const totalSupply = await client.token.getTotalSupply({
  token: '0x20c0000000000000000000000000000000000000',
})

console.log('Total Supply:', totalSupply)
// @log: Total Supply: { amount: 1000000000000000n, decimals: 6, formatted: '1000000000' }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
// [!include ~/snippets/tempo/viem.config.ts:setup]
```

:::

## Return Type

```ts
type ReturnType = {
  /** Total supply in the token's base units. */
  amount: bigint
  /** Token decimals used to derive `formatted`. */
  decimals: number
  /** Total supply formatted as a human-readable decimal string. */
  formatted: string
}
```

The total supply, both in base units (`amount`) and as a human-readable decimal
string (`formatted`, derived from the token's `decimals`).

## Parameters

### decimals (optional)

- **Type:** `number`

Decimals used to convert between base units and the human-readable amount.
Inferred from the Client's `tokens` set when `token` matches a declared token;
otherwise fetched from the token contract when needed.

### token

- **Type:** `Address | bigint`

Token to operate on: a TIP-20 token id or a contract address.
