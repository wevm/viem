# token.getTotalSupply

Gets the total supply of a TIP-20 token.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const totalSupply = await client.token.getTotalSupply({
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { amount: 1000000000000n, decimals: 6, formatted: '1000000' }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const totalSupply = await Actions.token.getTotalSupply(client, {
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { amount: 1000000000000n, decimals: 6, formatted: '1000000' }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Parameters

### decimals

- **Type:** `number`

Decimals used to convert between base units and the human-readable amount. Inferred from the client's `tokens` array when `token` matches a declared token; otherwise fetched from the token contract when needed.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const totalSupply = await Actions.token.getTotalSupply(client, {
  decimals: 6, // [!code focus]
  token: '0x20c0000000000000000000000000000000000001',
})
```

### token

- **Type:** `TokenId | Address`

Address or ID of the TIP-20 token.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const totalSupply = await Actions.token.getTotalSupply(client, {
  token: 1n, // [!code focus]
})
```

## Return Value

`{ amount: bigint, decimals: number, formatted: string }`

The total supply, in base units (`amount`) and human-readable form (`formatted`, derived from `decimals`).

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The token read failed. |
