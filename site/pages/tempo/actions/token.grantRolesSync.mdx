# token.grantRolesSync

Grants roles for a TIP-20 token, and waits for the transaction to be confirmed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { receipt, value } = await client.token.grantRolesSync({
  roles: ['issuer'],
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { receipt: { … }, value: [{ role: '0x…', account: '0x…', sender: '0x…', hasRole: true }] }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const { receipt, value } = await Actions.token.grantRolesSync(client, {
  roles: ['issuer'],
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { receipt: { … }, value: [{ role: '0x…', account: '0x…', sender: '0x…', hasRole: true }] }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),

```

:::

## Parameters

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### roles

- **Type:** `readonly TokenRole[]`

Roles to grant. Valid roles: `'defaultAdmin'`, `'pause'`, `'unpause'`, `'issuer'`, `'burnBlocked'`.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.token.grantRolesSync(client, {
  roles: ['issuer', 'pause'], // [!code focus]
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
```

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

### to

- **Type:** `Address`

Address to grant the role to.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.token.grantRolesSync(client, {
  roles: ['issuer'],
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
  token: '0x20c0000000000000000000000000000000000001',
})
```

### token

- **Type:** `TokenId | Address`

Address or ID of the TIP-20 token.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.token.grantRolesSync(client, {
  roles: ['issuer'],
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: 1n, // [!code focus]
})
```

## Return Value

`{ receipt: TransactionReceipt, value: readonly { role: Hex, account: Address, sender: Address, hasRole: boolean }[] }`

The transaction receipt and the `RoleMembershipUpdated` event data.

## Errors

| Error | Description |
| --- | --- |
| `RpcError.ExecutionError` | The transaction failed to execute. |
| `TransactionReceipt.RevertedError` | The transaction receipt status was `reverted` and `throwOnReceiptRevert` was enabled. |
