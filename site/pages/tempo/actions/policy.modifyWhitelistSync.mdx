# policy.modifyWhitelistSync

Modifies a policy whitelist, and waits for the transaction to be confirmed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { account, allowed } = await client.policy.modifyWhitelistSync({
  address: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  allowed: true,
  policyId: 2n,
})
// @log: { policyId: 2n, updater: '0x…', account: '0x…', allowed: true, receipt: { … } }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const { account, allowed } = await Actions.policy.modifyWhitelistSync(client, {
  address: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  allowed: true,
  policyId: 2n,
})
// @log: { policyId: 2n, updater: '0x…', account: '0x…', allowed: true, receipt: { … } }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### address

- **Type:** `Address`

Target account address.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.policy.modifyWhitelistSync(client, {
  address: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8', // [!code focus]
  allowed: true,
  policyId: 2n,
})
```

### allowed

- **Type:** `boolean`

Whether the account is allowed.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.policy.modifyWhitelistSync(client, {
  address: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  allowed: false, // [!code focus]
  policyId: 2n,
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### policyId

- **Type:** `bigint`

Policy ID.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.policy.modifyWhitelistSync(client, {
  address: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  allowed: true,
  policyId: 2n, // [!code focus]
})
```

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

## Return Value

`{ policyId: bigint, updater: Address, account: Address, allowed: boolean, receipt: TransactionReceipt }`

The `WhitelistUpdated` event data and the transaction receipt.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed or the receipt reverted. |
