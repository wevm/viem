# dex.cancel

Cancels an order from the orderbook.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const hash = await client.dex.cancel({ orderId: 123n })
// @log: '0x…'
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const hash = await Actions.dex.cancel(client, { orderId: 123n })
// @log: '0x…'
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### orderId

- **Type:** `bigint`

Order ID to cancel.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.dex.cancel(client, {
  orderId: 123n, // [!code focus]
})
```

## Return Value

`Hex`

The transaction hash.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed (for example, the caller does not own the order). |
