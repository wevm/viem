# token.transfer

Transfers TIP-20 tokens to another address.

Pass `from` to transfer on behalf of another address using an allowance (calls `transferFrom`); otherwise transfers from the caller (calls `transfer`).

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const hash = await client.token.transfer({
  amount: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: '0x…'
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const hash = await Actions.token.transfer(client, {
  amount: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: '0x…'
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### amount

- **Type:** `bigint | { decimals?: number, formatted: string }`

Amount of tokens to transfer, in base units or formatted decimal form.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.token.transfer(client, {
  amount: { decimals: 6, formatted: '1.5' }, // [!code focus]
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.token.transfer(client, {
  amount: 100n,
  feeToken: 1n, // [!code focus]
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
```

### from

- **Type:** `Address`

Address to transfer tokens from (uses an allowance via `transferFrom`).

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.token.transfer(client, {
  amount: 100n,
  from: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8', // [!code focus]
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
```

### memo

- **Type:** `Hex`

Memo to include in the transfer (calls `transferWithMemo`).

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.token.transfer(client, {
  amount: 100n,
  memo: '0xdeadbeef', // [!code focus]
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
```

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.token.transfer(client, {
  amount: 100n,
  nonceKey: 'random', // [!code focus]
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
```

### to

- **Type:** `Address`

Address to transfer tokens to.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.token.transfer(client, {
  amount: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
  token: '0x20c0000000000000000000000000000000000001',
})
```

### token

- **Type:** `TokenId | Address`

Address or ID of the TIP-20 token.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.token.transfer(client, {
  amount: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: 1n, // [!code focus]
})
```

## Return Value

`Hex`

The transaction hash.
