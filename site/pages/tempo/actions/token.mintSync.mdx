# token.mintSync

Mints TIP-20 tokens to an address, and waits for the transaction to be confirmed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { to, amount, receipt } = await client.token.mintSync({
  amount: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { to: '0x…', amount: 100n, receipt: { … } }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const { to, amount, receipt } = await Actions.token.mintSync(client, {
  amount: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { to: '0x…', amount: 100n, receipt: { … } }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### amount

- **Type:** `bigint | { decimals?: number, formatted: string }`

Amount of tokens to mint, in base units or formatted decimal form.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.token.mintSync(client, {
  amount: { decimals: 6, formatted: '1.5' }, // [!code focus]
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### memo

- **Type:** `Hex`

Memo to include in the mint (calls `mintWithMemo`).

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.token.mintSync(client, {
  amount: 100n,
  memo: '0xdeadbeef', // [!code focus]
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: '0x20c0000000000000000000000000000000000001',
})
```

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

### to

- **Type:** `Address`

Address to mint tokens to.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.token.mintSync(client, {
  amount: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
  token: '0x20c0000000000000000000000000000000000001',
})
```

### token

- **Type:** `TokenId | Address`

Address or ID of the TIP-20 token.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.token.mintSync(client, {
  amount: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  token: 1n, // [!code focus]
})
```

## Return Value

`{ to: Address, amount: bigint, receipt: TransactionReceipt }`

The `Mint` event data and the transaction receipt.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed or the receipt reverted. |
