# dex.placeFlipSync

Places a flip order that automatically flips when filled, and waits for the transaction to be confirmed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { orderId, receipt } = await client.dex.placeFlipSync({
  amount: 100n,
  flipTick: 10,
  tick: -10,
  token: '0x20c0000000000000000000000000000000000001',
  type: 'buy',
})
// @log: {
// @log:   orderId: 123n,
// @log:   maker: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
// @log:   token: '0x20c0000000000000000000000000000000000001',
// @log:   amount: 100n,
// @log:   isBid: true,
// @log:   tick: -10,
// @log:   isFlipOrder: true,
// @log:   flipTick: 10,
// @log:   receipt: { … },
// @log: }
```

## Parameters

### amount

- **Type:** `bigint`

Amount of tokens to place in the order.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.dex.placeFlipSync(client, {
  amount: 100n, // [!code focus]
  flipTick: 10,
  tick: -10,
  token: '0x20c0000000000000000000000000000000000001',
  type: 'buy',
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### flipTick

- **Type:** `number`

Target tick to flip to when the order is filled.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.dex.placeFlipSync(client, {
  amount: 100n,
  flipTick: 10, // [!code focus]
  tick: -10,
  token: '0x20c0000000000000000000000000000000000001',
  type: 'buy',
})
```

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

### tick

- **Type:** `number`

Price tick for the order, between `-2000` and `2000` (tick `0` is a price of `1.0`). Use `Tick.fromPrice` from `'viem/tempo'` to convert a price into a tick.

```ts twoslash
import { Actions, Tick } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.dex.placeFlipSync(client, {
  amount: 100n,
  flipTick: 10,
  tick: Tick.fromPrice('0.999'), // [!code focus]
  token: '0x20c0000000000000000000000000000000000001',
  type: 'buy',
})
```

### token

- **Type:** `TokenId | Address`

Address or ID of the base token.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.dex.placeFlipSync(client, {
  amount: 100n,
  flipTick: 10,
  tick: -10,
  token: 1n, // [!code focus]
  type: 'buy',
})
```

### type

- **Type:** `'buy' | 'sell'`

Order type: `buy` to buy the token, `sell` to sell it.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.dex.placeFlipSync(client, {
  amount: 100n,
  flipTick: -10,
  tick: 10,
  token: '0x20c0000000000000000000000000000000000001',
  type: 'sell', // [!code focus]
})
```

## Return Value

`{ orderId: bigint, maker: Address, token: Address, amount: bigint, isBid: boolean, tick: number, isFlipOrder: boolean, flipTick: number, receipt: TransactionReceipt }`

The `OrderPlaced` event data and the transaction receipt.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed or the receipt reverted. |
