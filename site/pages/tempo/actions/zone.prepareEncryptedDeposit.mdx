# `zone.prepareEncryptedDeposit`

Prepares an encrypted zone deposit payload without broadcasting a transaction.
Use this when one party should choose the private zone recipient, but another
party will submit the parent-chain deposit transaction.

:::info
If the same client should prepare and broadcast the deposit in one step, use
[`zone.encryptedDeposit`](/tempo/actions/zone.encryptedDeposit) instead.
:::

:::warning
Prepared encrypted deposits contain public transaction intent. If you receive a prepared payload from
another party, validate `chainId`, `zoneId`, `portalAddress`, `token`, and `amount` before passing it
to [`zone.encryptedDeposit`](/tempo/actions/zone.encryptedDeposit).
:::

## Usage

:::code-group

```ts twoslash [prepare.ts]
import { parseUnits } from 'viem'
import { broadcaster, client } from './viem.config'

const prepared = await client.zone.prepareEncryptedDeposit({
  amount: parseUnits('100', 6),
  recipient: '0x0000000000000000000000000000000000000001',
  token: '0x20c0000000000000000000000000000000000001',
  zoneId: 7,
})

const hash = await broadcaster.zone.encryptedDeposit(prepared)
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
// [!include ~/snippets/tempo/viem.config.ts:setup]

export const broadcaster = client
```

:::

## Return Type

```ts
type ReturnType = {
  /** Amount of tokens to deposit. */
  amount: bigint
  /** Parent chain ID. */
  chainId: number
  /** Encrypted deposit payload. */
  encrypted: {
    ciphertext: Hex
    ephemeralPubkeyX: Hex
    ephemeralPubkeyYParity: number
    nonce: Hex
    tag: Hex
  }
  /** Encryption key index from the portal contract. */
  keyIndex: bigint
  /** Zone portal address on the parent chain. */
  portalAddress: Address
  /** Token address or ID to deposit. */
  token: Address | bigint
  /** Zone ID. */
  zoneId: number
}
```

## Parameters

### amount

- **Type:** `bigint`

Amount of tokens to deposit.

### recipient

- **Type:** `Address`

Recipient address in the zone.

### token

- **Type:** `Address | bigint`

Token address or ID to deposit.

### zoneId

- **Type:** `number`

Zone ID to deposit into (e.g. `7`).

### memo (optional)

- **Type:** `Hex`
- **Default:** `0x00...00`

Optional deposit memo.
