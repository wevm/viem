# dex.place

Places a limit order on the orderbook. Prices are expressed as ticks: use the `Tick` helpers from `'viem/tempo'` to convert between prices and ticks.

## Usage

:::code-group

```ts twoslash [Decorator]
import { Tick } from 'viem/tempo'
import { client } from './viem.config'

const hash = await client.dex.place({
  amount: 100n,
  tick: Tick.fromPrice('0.999'),
  token: '0x20c0000000000000000000000000000000000001',
  type: 'buy',
})
// @log: '0x…'
```

```ts twoslash [Standalone]
import { Actions, Tick } from 'viem/tempo'
import { client } from './viem.config'

const hash = await Actions.dex.place(client, {
  amount: 100n,
  tick: Tick.fromPrice('0.999'),
  token: '0x20c0000000000000000000000000000000000001',
  type: 'buy',
})
// @log: '0x…'
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),

```

:::

## Parameters

### amount

- **Type:** `bigint`

Amount of tokens to place in the order.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.dex.place(client, {
  amount: 100n, // [!code focus]
  tick: -10,
  token: '0x20c0000000000000000000000000000000000001',
  type: 'buy',
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### tick

- **Type:** `number`

Price tick for the order, between `-2000` and `2000` (tick `0` is a price of `1.0`). Use `Tick.fromPrice` from `'viem/tempo'` to convert a price into a tick.

```ts twoslash
import { Actions, Tick } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.dex.place(client, {
  amount: 100n,
  tick: Tick.fromPrice('0.999'), // [!code focus]
  token: '0x20c0000000000000000000000000000000000001',
  type: 'buy',
})
```

### token

- **Type:** `TokenId | Address`

Address or ID of the base token.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.dex.place(client, {
  amount: 100n,
  tick: -10,
  token: 1n, // [!code focus]
  type: 'buy',
})
```

### type

- **Type:** `'buy' | 'sell'`

Order type: `buy` to buy the token, `sell` to sell it.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.dex.place(client, {
  amount: 100n,
  tick: 10,
  token: '0x20c0000000000000000000000000000000000001',
  type: 'sell', // [!code focus]
})
```

## Return Value

`Hex`

The transaction hash.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed (for example, the tick is out of bounds). |
