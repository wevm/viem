# channel.topUpSync

Adds deposit to a TIP-20 payment channel, and waits for the transaction to be confirmed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { newDeposit, receipt } = await client.channel.topUpSync({
  additionalDeposit: 100n,
  channel: {
    expiringNonceHash: '0x…',
    payee: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
    payer: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
    salt: '0x…',
    token: 1n,
  },
})
// @log: {
// @log:   channelId: '0x…',
// @log:   payer: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
// @log:   payee: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
// @log:   additionalDeposit: 100n,
// @log:   newDeposit: 200n,
// @log:   receipt: { … },
// @log: }
```

## Parameters

### additionalDeposit

- **Type:** `bigint`

Additional deposit to lock in the channel.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const channel = {
  expiringNonceHash: '0x…',
  payee: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  payer: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  salt: '0x…',
  token: 1n,
} as const
// ---cut---
const result = await Actions.channel.topUpSync(client, {
  additionalDeposit: 100n, // [!code focus]
  channel,
})
```

### channel

- **Type:** `Channel`

Channel descriptor: the fields that identify the channel (`payer`, `payee`, `token`, `salt`, `expiringNonceHash`, plus optional `operator` and `authorizedSigner`). Use the event data returned by [`channel.openSync`](/tempo/actions/channel.openSync).

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.channel.topUpSync(client, {
  additionalDeposit: 100n,
  channel: { // [!code focus]
    expiringNonceHash: '0x…', // [!code focus]
    payee: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8', // [!code focus]
    payer: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
    salt: '0x…', // [!code focus]
    token: 1n, // [!code focus]
  }, // [!code focus]
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

## Return Value

`{ channelId: Hex, payer: Address, payee: Address, additionalDeposit: bigint, newDeposit: bigint, receipt: TransactionReceipt }`

The `TopUp` event data and the transaction receipt. `newDeposit` is the total deposit after the top up.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed or the receipt reverted. |
