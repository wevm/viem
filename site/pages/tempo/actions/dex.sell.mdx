# dex.sell

Sells a specific amount of tokens.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const hash = await client.dex.sell({
  amountIn: 100n,
  minAmountOut: 95n,
  tokenIn: '0x20c0000000000000000000000000000000000001',
  tokenOut: '0x20c0000000000000000000000000000000000002',
})
// @log: '0x…'
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const hash = await Actions.dex.sell(client, {
  amountIn: 100n,
  minAmountOut: 95n,
  tokenIn: '0x20c0000000000000000000000000000000000001',
  tokenOut: '0x20c0000000000000000000000000000000000002',
})
// @log: '0x…'
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### amountIn

- **Type:** `bigint`

Amount of `tokenIn` to sell.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.dex.sell(client, {
  amountIn: 100n, // [!code focus]
  minAmountOut: 95n,
  tokenIn: '0x20c0000000000000000000000000000000000001',
  tokenOut: '0x20c0000000000000000000000000000000000002',
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### minAmountOut

- **Type:** `bigint`

Minimum amount of `tokenOut` to receive.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.dex.sell(client, {
  amountIn: 100n,
  minAmountOut: 95n, // [!code focus]
  tokenIn: '0x20c0000000000000000000000000000000000001',
  tokenOut: '0x20c0000000000000000000000000000000000002',
})
```

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### tokenIn

- **Type:** `TokenId | Address`

Address or ID of the token to sell.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.dex.sell(client, {
  amountIn: 100n,
  minAmountOut: 95n,
  tokenIn: 1n, // [!code focus]
  tokenOut: '0x20c0000000000000000000000000000000000002',
})
```

### tokenOut

- **Type:** `TokenId | Address`

Address or ID of the token to receive.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.dex.sell(client, {
  amountIn: 100n,
  minAmountOut: 95n,
  tokenIn: '0x20c0000000000000000000000000000000000001',
  tokenOut: 2n, // [!code focus]
})
```

## Return Value

`Hex`

The transaction hash.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed (for example, the swap would return less than `minAmountOut`). |
