# token.hasRole

Checks if an account has a specific role for a TIP-20 token.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const hasRole = await client.token.hasRole({
  account: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  role: 'issuer',
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: true
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const hasRole = await Actions.token.hasRole(client, {
  account: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  role: 'issuer',
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: true
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### account

- **Type:** `Account | Address`
- **Default:** `client.account`

Account (or address) to check.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hasRole = await Actions.token.hasRole(client, {
  account: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
  role: 'issuer',
  token: '0x20c0000000000000000000000000000000000001',
})
```

### role

- **Type:** `TokenRole`

Role to check. Valid roles: `'defaultAdmin'`, `'pause'`, `'unpause'`, `'issuer'`, `'burnBlocked'`.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hasRole = await Actions.token.hasRole(client, {
  account: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  role: 'pause', // [!code focus]
  token: '0x20c0000000000000000000000000000000000001',
})
```

### token

- **Type:** `TokenId | Address`

Address or ID of the TIP-20 token.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hasRole = await Actions.token.hasRole(client, {
  account: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  role: 'issuer',
  token: 1n, // [!code focus]
})
```

## Return Value

`boolean`

Whether the account has the role.

## Errors

| Error | Description |
| --- | --- |
| `Account.NotFoundError` | No `account` was provided and the client has no account. |
| `ContractFunctionExecutionError` | The token read failed. |
