# amm.mint

Adds liquidity to a fee AMM pool. The fee AMM is single-sided: liquidity is provided in the validator token.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const hash = await client.amm.mint({
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  userTokenAddress: '0x20c0000000000000000000000000000000000001',
  validatorTokenAddress: '0x20c0000000000000000000000000000000000002',
  validatorTokenAmount: 100n,
})
// @log: '0x…'
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const hash = await Actions.amm.mint(client, {
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  userTokenAddress: '0x20c0000000000000000000000000000000000001',
  validatorTokenAddress: '0x20c0000000000000000000000000000000000002',
  validatorTokenAmount: 100n,
})
// @log: '0x…'
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'

  account: Account.fromPrivateKey('0x…'),
```

:::

## Parameters

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### to

- **Type:** `Address`

Address to mint LP tokens to.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.amm.mint(client, {
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
  userTokenAddress: '0x20c0000000000000000000000000000000000001',
  validatorTokenAddress: '0x20c0000000000000000000000000000000000002',
  validatorTokenAmount: 100n,
})
```

### userTokenAddress

- **Type:** `TokenId | Address`

Address or ID of the user token of the pool.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.amm.mint(client, {
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  userTokenAddress: 1n, // [!code focus]
  validatorTokenAddress: '0x20c0000000000000000000000000000000000002',
  validatorTokenAmount: 100n,
})
```

### validatorTokenAddress

- **Type:** `TokenId | Address`

Address or ID of the validator token of the pool.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.amm.mint(client, {

  userTokenAddress: '0x20c0000000000000000000000000000000000001',
  validatorTokenAddress: 2n, // [!code focus]
  validatorTokenAmount: 100n,
})
```

### validatorTokenAmount

- **Type:** `bigint`

Amount of validator token to add.

```ts twoslash

import { client } from './viem.config'
// ---cut---

```
