# receivePolicy.getBlockedBalance

Gets the blocked balance for an encoded receipt: the amount held by the `ReceivePolicyGuard` that the receipt can claim or burn.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const amount = await client.receivePolicy.getBlockedBalance({
  receipt: '0x…',
})
// @log: 100n
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const amount = await Actions.receivePolicy.getBlockedBalance(client, {
  receipt: '0x…',
})
// @log: 100n
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Parameters

### receipt

- **Type:** `ReceivePolicyReceipt`

The encoded claim receipt (witness from a `TransferBlocked` event). The `ReceivePolicyReceipt` type is re-exported from `'viem/tempo'`.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const amount = await Actions.receivePolicy.getBlockedBalance(client, {
  receipt: '0x…', // [!code focus]
})
```

## Return Value

`bigint`

The blocked amount for the receipt, in base units.

## Errors

| Error | Description |
| --- | --- |
