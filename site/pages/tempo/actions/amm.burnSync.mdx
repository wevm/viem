# amm.burnSync

Removes liquidity from a fee AMM pool, and waits for the transaction to be confirmed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { amountUserToken, amountValidatorToken, receipt } =
  await client.amm.burnSync({
    liquidity: 50n,
    to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
    userToken: '0x20c0000000000000000000000000000000000001',
    validatorToken: '0x20c0000000000000000000000000000000000002',
  })
// @log: {
// @log:   sender: '0x…',
// @log:   userToken: '0x20c0000000000000000000000000000000000001',
// @log:   validatorToken: '0x20c0000000000000000000000000000000000002',
// @log:   amountUserToken: 25n,
// @log:   amountValidatorToken: 25n,
// @log:   liquidity: 50n,
// @log:   to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
// @log:   receipt: { … },
// @log: }
```

## Parameters

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### liquidity

- **Type:** `bigint`

Amount of LP tokens to burn.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.amm.burnSync(client, {
  liquidity: 50n, // [!code focus]
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  userToken: '0x20c0000000000000000000000000000000000001',
  validatorToken: '0x20c0000000000000000000000000000000000002',
})
```

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

### to

- **Type:** `Address`

Address to send the withdrawn tokens to.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.amm.burnSync(client, {
  liquidity: 50n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
  userToken: '0x20c0000000000000000000000000000000000001',
  validatorToken: '0x20c0000000000000000000000000000000000002',
})
```

### userToken

- **Type:** `TokenId | Address`

Address or ID of the user token of the pool.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.amm.burnSync(client, {
  liquidity: 50n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  userToken: 1n, // [!code focus]
  validatorToken: '0x20c0000000000000000000000000000000000002',
})
```

### validatorToken

- **Type:** `TokenId | Address`

Address or ID of the validator token of the pool.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.amm.burnSync(client, {
  liquidity: 50n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  userToken: '0x20c0000000000000000000000000000000000001',
  validatorToken: 2n, // [!code focus]
})
```

## Return Value

`{ sender: Address, userToken: Address, validatorToken: Address, amountUserToken: bigint, amountValidatorToken: bigint, liquidity: bigint, to: Address, receipt: TransactionReceipt }`

The `Burn` event data and the transaction receipt.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed or the receipt reverted. |
