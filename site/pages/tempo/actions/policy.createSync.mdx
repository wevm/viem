# policy.createSync

Creates a new TIP-403 transfer policy, and waits for the transaction to be confirmed. Returns the new policy's ID from the `PolicyCreated` event.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { policyId } = await client.policy.createSync({
  type: 'whitelist',
})
// @log: { policyId: 2n, updater: '0x…', policyType: 0, receipt: { … } }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const { policyId } = await Actions.policy.createSync(client, {
  type: 'whitelist',
})
// @log: { policyId: 2n, updater: '0x…', policyType: 0, receipt: { … } }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### addresses

- **Type:** `readonly Address[]`

Accounts to initialize the policy with.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const { policyId } = await Actions.policy.createSync(client, {
  addresses: [ // [!code focus]
    '0x70997970C51812dc3A010C7d01b50e0d17dc79C8', // [!code focus]
  ], // [!code focus]
  type: 'whitelist',
})
```

### admin

- **Type:** `Account | Address`
- **Default:** `client.account`

Policy admin (or address).

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const { policyId } = await Actions.policy.createSync(client, {
  admin: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
  type: 'whitelist',
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

### type

- **Type:** `'whitelist' | 'blacklist'`

Type of policy to create.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const { policyId } = await Actions.policy.createSync(client, {
  type: 'blacklist', // [!code focus]
})
```

## Return Value

`{ policyId: bigint, updater: Address, policyType: number, receipt: TransactionReceipt }`

The `PolicyCreated` event data and the transaction receipt. `policyType` is `0` for whitelist, `1` for blacklist.

## Errors

| Error | Description |
| --- | --- |
| `BaseError` | No `admin` was provided and the client has no account. |
| `ContractFunctionExecutionError` | The write failed or the receipt reverted. |
