# token.getBalance

Gets TIP-20 token balance for an address.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const balance = await client.token.getBalance({
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { amount: 100000000n, decimals: 6, formatted: '100' }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const balance = await Actions.token.getBalance(client, {
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { amount: 100000000n, decimals: 6, formatted: '100' }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### account

- **Type:** `Account | Address`
- **Default:** `client.account`

Account (or address) that owns the tokens.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const balance = await Actions.token.getBalance(client, {
  account: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
  token: '0x20c0000000000000000000000000000000000001',
})
```

### decimals

- **Type:** `number`

Decimals used to convert between base units and the human-readable amount. Inferred from the client's `tokens` array when `token` matches a declared token; otherwise fetched from the token contract when needed.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const balance = await Actions.token.getBalance(client, {
  decimals: 6, // [!code focus]
  token: '0x20c0000000000000000000000000000000000001',
})
```

### token

- **Type:** `TokenId | Address`

Address or ID of the TIP-20 token.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const balance = await Actions.token.getBalance(client, {
  token: 1n, // [!code focus]
})
```

## Return Value

`{ amount: bigint, decimals: number, formatted: string }`

The balance, in base units (`amount`) and human-readable form (`formatted`, derived from `decimals`).

## Errors

| Error | Description |
| --- | --- |
| `Account.NotFoundError` | No `account` was provided and the client has no account. |
| `ContractFunctionExecutionError` | The token read failed. |
