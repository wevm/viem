# `zone.getDepositStatus`

Returns deposit processing status for a given parent Tempo chain block number.

## Usage

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

const status = await client.zone.getDepositStatus({
  tempoBlockNumber: 42n,
})

console.log('Processed:', status.processed)
console.log('Deposits:', status.deposits)
// @log: Processed: true
// @log: Deposits: [{ amount: 100000000n, kind: 'regular', status: 'processed', ... }]
```

## Return Value

```ts
type ReturnType = {
  /** Deposits included in the block. */
  deposits: readonly {
    amount: bigint
    depositHash: Hex
    kind: 'encrypted' | 'regular'
    memo: Hex | null
    recipient: Address | null
    sender: Address
    status: 'failed' | 'pending' | 'processed'
    token: Address
  }[]
  /** Whether the block has been fully processed by the zone. */
  processed: boolean
  /** Queried parent chain block number. */
  tempoBlockNumber: bigint
  /** Latest parent chain block the zone has processed. */
  zoneProcessedThrough: bigint
}
```

The deposit processing status.

## Parameters

### tempoBlockNumber

- **Type:** `bigint`

Parent Tempo chain block number to query.
