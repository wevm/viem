# `channel.settle`

Settles a signed TIP-20 channel reserve voucher and pays the delta to the payee.

## Usage

:::code-group

```ts twoslash [example.ts]
import { parseUnits } from 'viem'
import { client, channel } from './viem.config'

const { deltaPaid, newSettled, receipt } =
  await client.channel.settleSync({
    cumulativeAmount: parseUnits('40', 6),
    channel,
    signature: '0x...',
  })

console.log('Delta paid:', deltaPaid)
console.log('New settled:', newSettled)
console.log('Transaction hash:', receipt.transactionHash)
```


:::

### Voucher Signature

The `signature` should be produced by the payer, or the channel's `authorizedSigner`.

```ts twoslash
import { parseUnits } from 'viem'
import { client, channel } from './viem.config'

const signature = await client.channel.signVoucher({
  channel,
  cumulativeAmount: parseUnits('40', 6),
})
```

### Asynchronous Usage

The example above uses a `*Sync` variant of the action, that will wait for the transaction to be included before returning.

If you are optimizing for performance, you should use the non-sync `channel.settle` action and wait for inclusion manually:

```ts twoslash
import { Actions } from 'viem/tempo'
import { client, channel } from './viem.config'

const hash = await client.channel.settle({
  cumulativeAmount: 40000000n,
  channel,
  signature: '0x...',
})
const receipt = await client.waitForTransactionReceipt({ hash })

const { args } = Actions.channel.settle.extractEvent(receipt.logs)
```

## Return Type

```ts
type ReturnType = {
  /** Channel ID */
  channelId: Hex
  /** Total voucher amount signed for the channel */
  cumulativeAmount: bigint
  /** Amount paid in this settlement */
  deltaPaid: bigint
  /** New total amount settled to the payee */
  newSettled: bigint
  /** Account that receives settled voucher payments */
  payee: Address
  /** Account that funded the channel and receives refunds */
  payer: Address
  /** Transaction receipt */
  receipt: TransactionReceipt
}
```

## Parameters

### cumulativeAmount

- **Type:** `bigint`

Total voucher amount signed for the channel.

### channel

- **Type:** `Channel.from.Value`

Channel containing the payer, payee, operator, token, salt, authorized signer, and expiring nonce hash.

### signature

- **Type:** `Hex`

Voucher signature for the cumulative amount.
