# `earn.getEarnVault`

Gets an EarnVault's configuration, accounting state, and supported actions.

## Usage

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

const earnVault = await client.earn.getEarnVault({
  earnVault: '0x0000000000000000000000000000000000000001',
})
```

## Return Value

```ts
type ReturnValue = {
  /** Token accepted by the EarnVault. */
  assetToken: Address
  /** Address allowed to cancel queued redemptions. */
  asyncJanitor: Address
  /** Actions supported by the current venue integration. */
  capabilities: {
    aredeem: boolean
    exactWithdraw: boolean
    inKindDeposit: boolean
    redeem: boolean
  }
  /** Whether new deposits are paused. */
  depositsPaused: boolean
  /** Address allowed to pause deposits. */
  emergencyGuardian: Address
  /** Current venue integration. */
  engine: {
    address: Address
    name: string
    symbol: string
    totalAssets: bigint
  }
  /** Who may change the venue integration. */
  engineMigrationMode: 'operatorEnabled' | 'userOnly'
  /** Venue shares held for the EarnVault. */
  engineShares: bigint
  /** Whether fees are active. */
  feesActive: boolean
  /** Whether EarnShare supply matches its asset backing. */
  isAccountingAligned: boolean
  /** EarnVault operator address. */
  operator: Address
  /** Open queued redemptions. */
  openRedeemRequestCount: bigint
  /** Active EarnShare supply. */
  totalEarnShares: bigint
  /** EarnShare token. */
  earnShare: Address
}
```

## Parameters

### earnVault

- **Type:** `Address`

EarnVault address.
