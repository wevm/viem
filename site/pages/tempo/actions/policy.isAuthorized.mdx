# policy.isAuthorized

Checks if a user is authorized by a policy.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const authorized = await client.policy.isAuthorized({
  policyId: 2n,
  user: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
})
// @log: true
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const authorized = await Actions.policy.isAuthorized(client, {
  policyId: 2n,
  user: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
})
// @log: true
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Parameters

### policyId

- **Type:** `bigint`

Policy ID.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const authorized = await Actions.policy.isAuthorized(client, {
  policyId: 2n, // [!code focus]
  user: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
})
```

### user

- **Type:** `Address`

User address to check.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const authorized = await Actions.policy.isAuthorized(client, {
  policyId: 2n,
  user: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8', // [!code focus]
})
```

## Return Value

`boolean`

Whether the user is authorized by the policy.
