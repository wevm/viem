# dex.withdrawSync

Withdraws tokens from the DEX to the caller's wallet, and waits for the transaction to be confirmed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { receipt } = await client.dex.withdrawSync({
  amount: 100n,
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { receipt: { … } }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const { receipt } = await Actions.dex.withdrawSync(client, {
  amount: 100n,
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { receipt: { … } }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### amount

- **Type:** `bigint`

Amount to withdraw.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const { receipt } = await Actions.dex.withdrawSync(client, {
  amount: 100n, // [!code focus]
  token: '0x20c0000000000000000000000000000000000001',
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

### token

- **Type:** `TokenId | Address`

Address or ID of the token to withdraw.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const { receipt } = await Actions.dex.withdrawSync(client, {
  amount: 100n,
  token: 1n, // [!code focus]
})
```

## Return Value

`{ receipt: TransactionReceipt }`

The transaction receipt.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed or the receipt reverted. |
