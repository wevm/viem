# token.updateQuoteTokenSync

Completes a quote token update for a TIP-20 token, and waits for the transaction to be confirmed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { updater, newQuoteToken, receipt } = await client.token.updateQuoteTokenSync({
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { updater: '0x…', newQuoteToken: '0x…', receipt: { … } }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const { updater, newQuoteToken, receipt } = await Actions.token.updateQuoteTokenSync(client, {
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { updater: '0x…', newQuoteToken: '0x…', receipt: { … } }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

### token

- **Type:** `TokenId | Address`

Address or ID of the TIP-20 token.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.token.updateQuoteTokenSync(client, {
  token: 1n, // [!code focus]
})
```

## Return Value

`{ updater: Address, newQuoteToken: Address, receipt: TransactionReceipt }`

The `QuoteTokenUpdate` event data and the transaction receipt.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed or the receipt reverted. |
