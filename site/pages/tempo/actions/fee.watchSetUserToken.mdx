# fee.watchSetUserToken

Watches for user fee-token updates, returning a watcher handle.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const watcher = client.fee.watchSetUserToken()
watcher.onLogs((logs) => console.log(logs))
// later: watcher.off()
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const watcher = Actions.fee.watchSetUserToken(client)
watcher.onLogs((logs) => console.log(logs))
// later: watcher.off()
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Return Value

`Watcher`

A watcher handle over `UserTokenSet` events: register listeners with `onLogs`/`onError`, stop with `off`, or consume it as an async iterable.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The event subscription failed. |
