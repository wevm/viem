# token.setSupplyCapSync

Sets the supply cap for a TIP-20 token, and waits for the transaction to be confirmed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { updater, newSupplyCap, receipt } = await client.token.setSupplyCapSync({
  supplyCap: 1000000n,
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { updater: '0x…', newSupplyCap: 1000000n, receipt: { … } }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const { updater, newSupplyCap, receipt } = await Actions.token.setSupplyCapSync(client, {
  supplyCap: 1000000n,
  token: '0x20c0000000000000000000000000000000000001',
})
// @log: { updater: '0x…', newSupplyCap: 1000000n, receipt: { … } }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### supplyCap

- **Type:** `bigint`

New supply cap, in base units.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.token.setSupplyCapSync(client, {
  supplyCap: 2000000n, // [!code focus]
  token: '0x20c0000000000000000000000000000000000001',
})
```

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

### token

- **Type:** `TokenId | Address`

Address or ID of the TIP-20 token.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.token.setSupplyCapSync(client, {
  supplyCap: 1000000n,
  token: 1n, // [!code focus]
})
```

## Return Value

`{ updater: Address, newSupplyCap: bigint, receipt: TransactionReceipt }`

The `SupplyCapUpdate` event data and the transaction receipt.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed or the receipt reverted. |
