# validator.getNextFullDkgCeremony

Gets the next epoch for a full DKG (distributed key generation) ceremony.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const epoch = await client.validator.getNextFullDkgCeremony()
// @log: 100n
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const epoch = await Actions.validator.getNextFullDkgCeremony(client)
// @log: 100n
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Return Value

`bigint`

The epoch number for the next full DKG ceremony.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The ValidatorConfig read failed. |
