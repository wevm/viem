# `earn.getPosition`

Gets an account's asset and EarnShare balances, allowances, and current share value.

## Usage

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

const position = await client.earn.getPosition({
  earnVault: '0x0000000000000000000000000000000000000001',
})
// @log: {
//   assetAllowance: 0n,
//   assetBalance: 250_000_000n,
//   assetToken: '0x20c0...0001',
//   earnShareAllowance: 0n,
//   earnShareBalance: 99_500_000n,
//   earnShare: '0x20c0...0002',
//   value: 100_000_000n,
// }
```

### Read Another Account

Pass `account` to inspect a position other than the client account.

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

const position = await client.earn.getPosition({
  account: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEbb',
  earnVault: '0x0000000000000000000000000000000000000001',
})
// @log: {
//   assetAllowance: 0n,
```

## Return Value

```ts
type ReturnValue = {
  /** Assets the vault may spend from the account. */
  assetAllowance: bigint
  /** Asset balance held by the account. */
  assetBalance: bigint
  /** Token accepted by the earnVault. */
  assetToken: Address
  /** EarnShare the vault may spend from the account. */
  earnShareAllowance: bigint
  /** EarnShare balance held by the account. */
  earnShareBalance: bigint
  /** Token representing EarnShare. */
  earnShare: Address
  /** Current asset value of the EarnShare balance, including fees. */
  value: bigint
}
```

## Parameters

### account (optional)

- **Type:** `Account | Address`
- **Default:** `client.account`

Account whose position is returned.

### earnVault

- **Type:** `Address`

EarnVault address.
