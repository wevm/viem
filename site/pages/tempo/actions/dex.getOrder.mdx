# dex.getOrder

Gets an order's details from the orderbook.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const order = await client.dex.getOrder({ orderId: 123n })
// @log: {
// @log:   orderId: 123n,
// @log:   maker: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
// @log:   bookKey: '0x…',
// @log:   isBid: true,
// @log:   tick: -10,
// @log:   amount: 100n,
// @log:   remaining: 60n,
// @log:   prev: 0n,
// @log:   next: 124n,
// @log:   isFlip: false,
// @log:   flipTick: 0,
// @log: }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const order = await Actions.dex.getOrder(client, { orderId: 123n })
// @log: {
// @log:   orderId: 123n,
// @log:   maker: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
// @log:   bookKey: '0x…',
// @log:   isBid: true,
// @log:   tick: -10,
// @log:   amount: 100n,
// @log:   remaining: 60n,
// @log:   prev: 0n,
// @log:   next: 124n,
// @log:   isFlip: false,
// @log:   flipTick: 0,
// @log: }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Parameters

### orderId

- **Type:** `bigint`

Order ID to query.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const order = await Actions.dex.getOrder(client, {
  orderId: 123n, // [!code focus]
})
```

## Return Value

`{ orderId: bigint, maker: Address, bookKey: Hex, isBid: boolean, tick: number, amount: bigint, remaining: bigint, prev: bigint, next: bigint, isFlip: boolean, flipTick: number }`

The order details.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The orderbook read failed. |
