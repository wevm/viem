# amm.rebalanceSwap

Performs a rebalance swap from validator token to user token.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const hash = await client.amm.rebalanceSwap({
  amountOut: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  userToken: '0x20c0000000000000000000000000000000000001',
  validatorToken: '0x20c0000000000000000000000000000000000002',
})
// @log: '0x…'
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const hash = await Actions.amm.rebalanceSwap(client, {
  amountOut: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  userToken: '0x20c0000000000000000000000000000000000001',
  validatorToken: '0x20c0000000000000000000000000000000000002',
})
// @log: '0x…'
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### amountOut

- **Type:** `bigint`

Amount of user token to receive.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.amm.rebalanceSwap(client, {
  amountOut: 100n, // [!code focus]
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  userToken: '0x20c0000000000000000000000000000000000001',
  validatorToken: '0x20c0000000000000000000000000000000000002',
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### to

- **Type:** `Address`

Address to send the user token to.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.amm.rebalanceSwap(client, {
  amountOut: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
  userToken: '0x20c0000000000000000000000000000000000001',
  validatorToken: '0x20c0000000000000000000000000000000000002',
})
```

### userToken

- **Type:** `TokenId | Address`

Address or ID of the user token of the pool.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.amm.rebalanceSwap(client, {
  amountOut: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  userToken: 1n, // [!code focus]
  validatorToken: '0x20c0000000000000000000000000000000000002',
})
```

### validatorToken

- **Type:** `TokenId | Address`

Address or ID of the validator token of the pool.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.amm.rebalanceSwap(client, {
  amountOut: 100n,
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  userToken: '0x20c0000000000000000000000000000000000001',
  validatorToken: 2n, // [!code focus]
})
```

`Hex`

The transaction hash.
