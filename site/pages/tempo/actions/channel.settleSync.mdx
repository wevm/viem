# channel.settleSync

Settles a TIP-20 payment channel voucher, and waits for the transaction to be confirmed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const { deltaPaid, newSettled, receipt } = await client.channel.settleSync({
  channel: {
    expiringNonceHash: '0x…',
    payee: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
    payer: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
    salt: '0x…',
    token: 1n,
  },
  cumulativeAmount: 100n,
  signature: '0x…',
})
// @log: {
// @log:   channelId: '0x…',
// @log:   payer: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
// @log:   payee: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
// @log:   cumulativeAmount: 100n,
// @log:   deltaPaid: 40n,
// @log:   newSettled: 100n,
// @log:   receipt: { … },
// @log: }
```

## Parameters

### channel

- **Type:** `Channel`

Channel descriptor: the fields that identify the channel (`payer`, `payee`, `token`, `salt`, `expiringNonceHash`, plus optional `operator` and `authorizedSigner`). Use the event data returned by [`channel.openSync`](/tempo/actions/channel.openSync).

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const result = await Actions.channel.settleSync(client, {
  channel: { // [!code focus]
    expiringNonceHash: '0x…', // [!code focus]
    payee: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8', // [!code focus]
    payer: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
    salt: '0x…', // [!code focus]
    token: 1n, // [!code focus]
  }, // [!code focus]
  cumulativeAmount: 100n,
  signature: '0x…',
})
```

### cumulativeAmount

- **Type:** `bigint`

Total voucher amount signed for the channel.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const channel = {
  expiringNonceHash: '0x…',
  payee: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  payer: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  salt: '0x…',
  token: 1n,
} as const
// ---cut---
const result = await Actions.channel.settleSync(client, {
  channel,
  cumulativeAmount: 100n, // [!code focus]
  signature: '0x…',
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### signature

- **Type:** `Hex`

Voucher signature over `cumulativeAmount`. Produce it with [`channel.signVoucher`](/tempo/actions/channel.signVoucher).

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const channel = {

  payee: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  payer: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  salt: '0x…',
  token: 1n,
} as const
// ---cut---
const result = await Actions.channel.settleSync(client, {
  channel,
  cumulativeAmount: 100n,
  signature: '0x…', // [!code focus]
})
```

### throwOnReceiptRevert

- **Type:** `boolean`
- **Default:** `true`

Whether to throw when the receipt marks the transaction as reverted.

## Return Value

`{ channelId: Hex, payer: Address, payee: Address, cumulativeAmount: bigint, deltaPaid: bigint, newSettled: bigint, receipt: TransactionReceipt }`

The `Settled` event data and the transaction receipt. `deltaPaid` is the amount paid to the payee in this settlement; `newSettled` is the total settled so far.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed or the receipt reverted. |
