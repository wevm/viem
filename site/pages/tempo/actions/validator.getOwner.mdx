# validator.getOwner

Gets the owner of the `ValidatorConfig` precompile. The owner is the only account allowed to add validators, change validator status, and schedule DKG ceremonies.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const owner = await client.validator.getOwner()
// @log: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266'
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const owner = await Actions.validator.getOwner(client)
// @log: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266'
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Return Value

`Address`

The owner address.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The ValidatorConfig read failed. |
