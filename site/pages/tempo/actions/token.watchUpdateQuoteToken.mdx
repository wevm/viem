# token.watchUpdateQuoteToken

Watches for TIP-20 token quote token updates, returning a watcher handle.

Emits `NextQuoteTokenSet` logs when an update is proposed, and `QuoteTokenUpdate` logs when an update is completed.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const watcher = client.token.watchUpdateQuoteToken({
  token: '0x20c0000000000000000000000000000000000001',
})
watcher.onLogs((logs) => console.log(logs))
// later: watcher.off()
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const watcher = Actions.token.watchUpdateQuoteToken(client, {
  token: '0x20c0000000000000000000000000000000000001',
})
watcher.onLogs((logs) => console.log(logs))
// later: watcher.off()
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Parameters

### token

- **Type:** `TokenId | Address`

Address or ID of the TIP-20 token.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const watcher = Actions.token.watchUpdateQuoteToken(client, {
  token: 1n, // [!code focus]
})
```

## Return Value

`Watcher`

A watcher handle over `NextQuoteTokenSet` and `QuoteTokenUpdate` events: register listeners with `onLogs`/`onError`, stop with `off`, or consume it as an async iterable.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The event subscription failed. |
