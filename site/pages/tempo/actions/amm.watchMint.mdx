# amm.watchMint

Watches for liquidity mint events, returning a watcher handle.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const watcher = client.amm.watchMint()
watcher.onLogs((logs) => console.log(logs))
// later: watcher.off()
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const watcher = Actions.amm.watchMint(client)
watcher.onLogs((logs) => console.log(logs))
// later: watcher.off()
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client } from 'viem/tempo'

export const client = Client.create()
```

:::

## Parameters

### to

- **Type:** `Address`

Address of the recipient to filter events.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const watcher = Actions.amm.watchMint(client, {
  to: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
})
```

### userToken

- **Type:** `TokenId | Address`

Address or ID of the user token to filter events.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const watcher = Actions.amm.watchMint(client, {
  userToken: '0x20c0000000000000000000000000000000000001', // [!code focus]
})
```

### validatorToken

- **Type:** `TokenId | Address`

Address or ID of the validator token to filter events.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const watcher = Actions.amm.watchMint(client, {
  validatorToken: '0x20c0000000000000000000000000000000000002', // [!code focus]
})
```

## Return Value

`Watcher`

A watcher handle over `Mint` events: register listeners with `onLogs`/`onError`, stop with `off`, or consume it as an async iterable.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The event subscription failed. |
