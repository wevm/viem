# `zone.getWithdrawalFee`

Returns the fee required for a withdrawal from a zone, given a callback gas limit.

## Usage

:::code-group

```ts twoslash [example.ts]
import { client } from './zones.config'

const fee = await client.zone.getWithdrawalFee()

console.log('Fee:', fee)
// @log: Fee: 0n
```

```ts twoslash [zones.config.ts] filename="zones.config.ts"
// [!include ~/snippets/tempo/zones.config.ts:setup]
```

:::

### Callback Gas

```ts twoslash
import { client } from './zones.config'

const fee = await client.zone.getWithdrawalFee({
  callbackGas: 100_000n,
})
```

## Return Value

```ts
type ReturnType = bigint
```

The withdrawal fee.

## Parameters

### callbackGas

- **Type:** `bigint`
- **Default:** `0n`

Gas limit reserved for the withdrawal callback on the parent chain.
