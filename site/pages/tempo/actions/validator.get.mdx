# validator.get

Gets validator information by address.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const validator = await client.validator.get({
  validator: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
})
// @log: {
// @log:   publicKey: '0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef',
// @log:   active: true,
// @log:   index: 0n,
// @log:   validatorAddress: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
// @log:   inboundAddress: '192.168.1.1:8080',
// @log:   outboundAddress: '192.168.1.1:8080',
// @log: }
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const validator = await Actions.validator.get(client, {
  validator: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
})
// @log: {
// @log:   publicKey: '0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef',
// @log:   active: true,
// @log:   index: 0n,
// @log:   validatorAddress: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
// @log:   inboundAddress: '192.168.1.1:8080',
// @log:   outboundAddress: '192.168.1.1:8080',
// @log: }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"

```

:::

## Parameters

### validator

- **Type:** `Address`

Validator address.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const validator = await Actions.validator.get(client, {
  validator: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8', // [!code focus]
})
```

## Return Value

`{ publicKey: Hex, active: boolean, index: bigint, validatorAddress: Address, inboundAddress: string, outboundAddress: string }`

The validator information.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The ValidatorConfig read failed. |
