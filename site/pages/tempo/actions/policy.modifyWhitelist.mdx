# policy.modifyWhitelist

Modifies a policy whitelist, allowing or disallowing an account. Only the policy admin can modify a policy.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const hash = await client.policy.modifyWhitelist({
  address: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  allowed: true,
  policyId: 2n,
})
// @log: '0x…'
```

```ts twoslash [Standalone]
import { Actions } from 'viem/tempo'
import { client } from './viem.config'

const hash = await Actions.policy.modifyWhitelist(client, {
  address: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  allowed: true,
  policyId: 2n,
})
// @log: '0x…'
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account } from 'viem'
import { Client } from 'viem/tempo'

export const client = Client.create({
  account: Account.fromPrivateKey('0x…'),
})
```

:::

## Parameters

### address

- **Type:** `Address`

Target account address.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.policy.modifyWhitelist(client, {
  address: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8', // [!code focus]
  allowed: true,
  policyId: 2n,
})
```

### allowed

- **Type:** `boolean`

Whether the account is allowed.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.policy.modifyWhitelist(client, {
  address: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  allowed: false, // [!code focus]
  policyId: 2n,
})
```

### feeToken

- **Type:** `TokenId | Address`
- **Default:** `client.feeToken`

TIP-20 token to pay transaction fees with.

### nonceKey

- **Type:** `bigint | 'random'`

Nonce key to send the transaction under. Pass `'random'` to use a random key.

### policyId

- **Type:** `bigint`

Policy ID.

```ts twoslash
import { Actions } from 'viem/tempo'
import { client } from './viem.config'
// ---cut---
const hash = await Actions.policy.modifyWhitelist(client, {
  address: '0x70997970C51812dc3A010C7d01b50e0d17dc79C8',
  allowed: true,
  policyId: 2n, // [!code focus]
})
```

## Return Value

`Hex`

The transaction hash.

## Errors

| Error | Description |
| --- | --- |
| `ContractFunctionExecutionError` | The write failed (for example, the caller is not the policy admin or the policy is not a whitelist). |
