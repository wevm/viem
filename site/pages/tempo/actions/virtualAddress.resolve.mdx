# `virtualAddress.resolve`

Resolves a virtual address to its master address.

## Usage

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

const recipient = await client.virtualAddress.resolve({
  address: '0x0000000000000000000000000000000000000001',
})

console.log('Recipient:', recipient)
// @log: Recipient: 0x0000000000000000000000000000000000000001
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
// [!include ~/snippets/tempo/viem.config.ts:setup]
```

:::

## Return Value

```ts
type ReturnType = Address | null
```

The resolved address, or `null` if virtual and unregistered.

## Parameters

### address

- **Type:** `Address`

Address to resolve.
