# `Storage.memory`

Creates an in-memory storage backed by a `Map`. Useful for server-side environments or testing.

## Usage

```ts twoslash
import { Storage } from 'viem/tempo'

const storage = Storage.memory()

await storage.setItem('key', 'value')
const value = await storage.getItem('key')
// @log: 'value'
```

## Parameters

### options

- **Type:** `Storage.from.Options`
- **Default:** `{}`

Options for key prefixing.

### key

- **Type:** `string | undefined`

Key prefix prepended to all storage keys.

## Return Value

```ts
type ReturnType = Storage
```

A `Storage` instance backed by an in-memory `Map`.
