# Chain

## Overview

The **Chain** module describes an EVM chain: its id, name, native currency, RPC endpoints, block explorers, contracts, and optional encoding behavior.

A `Chain` is plain data plus two optional behavior slots:

- **`schema`**: bidirectional RPC to native codecs.
- **`serializers`**: transaction serialization overrides.

```ts twoslash
import { Chain } from 'viem'

const mainnet = Chain.from({
  id: 1,
  name: 'Ethereum',
  nativeCurrency: { name: 'Ether', symbol: 'ETH', decimals: 18 },
  rpcUrls: { default: { http: ['https://eth.merkle.io'] } },
})
```
