# contract.createEventFilter

Event filter creators, one per ABI event. Each maps to [`Actions.contract.createEventFilter`](/docs/actions/public/contract/createEventFilter) with `abi`, `address`, and `eventName` bound. Read matches later with [`Actions.filter.getChanges`](/docs/actions/public/filter/getChanges).

## Usage

```ts twoslash
import { Client, Contract, http } from 'viem'
import { mainnet } from 'viem/chains'
import { Abis } from 'viem/utils'

const client = Client.create({ chain: mainnet, transport: http() })

const contract = Contract.from({
  abi: Abis.erc20,
  address: '0xFBA3912Ca04dd458c843e2EE08967fC04f3579c2',
  client,
})

const filter = await contract.createEventFilter.Transfer({ // [!code focus]
  args: { from: '0xA0Cf798816D4b9b9866b5330EEa46a18382f251e' }, // [!code focus]
}) // [!code focus]
```

## Options

The options of [`Actions.contract.createEventFilter`](/docs/actions/public/contract/createEventFilter#parameters) minus the bound `abi`, `address`, and `eventName`, such as `args`, `fromBlock`, `toBlock`, and `strict`.

## Return Value

`Promise<Filter>`

The installed filter, typed against the event.
