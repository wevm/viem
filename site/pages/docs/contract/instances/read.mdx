# contract.read

Read-only (`pure`/`view`) contract methods, one per ABI function. Each maps to [`Actions.contract.read`](/docs/actions/public/contract/read) with `abi`, `address`, and `functionName` bound.

## Usage

```ts twoslash
import { Client, Contract, http } from 'viem'
import { mainnet } from 'viem/chains'
import { Abis } from 'viem/utils'

const client = Client.create({ chain: mainnet, transport: http() })

const contract = Contract.from({
  abi: Abis.erc20,
  address: '0xFBA3912Ca04dd458c843e2EE08967fC04f3579c2',
  client,
})

const balance = await contract.read.balanceOf([ // [!code focus]
  '0xA0Cf798816D4b9b9866b5330EEa46a18382f251e', // [!code focus]
]) // [!code focus]
// @log: 424122n
```

## Options

Function arguments are passed as the first parameter. The optional second parameter accepts the options of [`Actions.contract.read`](/docs/actions/public/contract/read#parameters) minus the bound `abi`, `address`, `args`, and `functionName`, such as `blockNumber` and `blockTag`.

When a function has no inputs, omit the arguments array and pass options as the first parameter:

```ts twoslash
import { Client, Contract, http } from 'viem'
import { mainnet } from 'viem/chains'
import { Abis } from 'viem/utils'

const client = Client.create({ chain: mainnet, transport: http() })

const contract = Contract.from({
  abi: Abis.erc20,
  address: '0xFBA3912Ca04dd458c843e2EE08967fC04f3579c2',
  client,
})
// ---cut---
const supply = await contract.read.totalSupply({
  blockTag: 'finalized', // [!code focus]
})
```

## Return Value

`Promise<ReadReturnType>`

The decoded return value of the function.
