# contract.estimateGas

Gas estimators for the ABI's state-mutating (`nonpayable`/`payable`) functions. Each maps to [`Actions.contract.estimateGas`](/docs/actions/public/contract/estimateGas) with `abi`, `address`, and `functionName` bound.

## Usage

```ts twoslash
import { Account, Client, Contract, http } from 'viem'
import { mainnet } from 'viem/chains'
import { Abis } from 'viem/utils'

const client = Client.create({
  account: Account.fromPrivateKey('0x...'),
  chain: mainnet,
  transport: http(),
})

const contract = Contract.from({
  abi: Abis.erc20,
  address: '0xFBA3912Ca04dd458c843e2EE08967fC04f3579c2',
  client,
})

const gas = await contract.estimateGas.transfer([ // [!code focus]
  '0x70997970c51812dc3a010c7d01b50e0d17dc79c8', // [!code focus]
  1n, // [!code focus]
]) // [!code focus]
// @log: 51441n
```

## Options

Function arguments are passed as the first parameter. The optional second parameter accepts the options of [`Actions.contract.estimateGas`](/docs/actions/public/contract/estimateGas#parameters) minus the bound `abi`, `address`, `args`, and `functionName`, such as `account` (to estimate as a specific sender).

When a function has no inputs, omit the arguments array and pass options as the first parameter.

## Return Value

`Promise<bigint>`

The estimated gas.
