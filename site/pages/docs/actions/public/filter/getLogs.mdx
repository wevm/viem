# getLogs

Returns the logs matching an event filter.

Unlike `getChanges`, this action is not drained by polling. It returns all logs held by the filter.

## Usage

:::code-group

```ts twoslash [Decorator]
import { AbiEvent } from 'ox'
import { Actions } from 'viem'
import { client } from './viem.config'

const transferEvent = AbiEvent.from(
  'event Transfer(address indexed from, address indexed to, uint256 value)',
)
// TODO: Replace with Actions.event.createFilter.
const id = await client.request({
  method: 'eth_newFilter',
  params: [
    {
      address: '0xfba3912ca04dd458c843e2ee08967fc04f3579c2',
      topics: AbiEvent.encode(transferEvent).topics,
    },
  ],
})
const filter = {
  abiEvent: transferEvent,
  id,
  request: client.request,
  type: 'event',
} satisfies Actions.filter.Filter<'event', typeof transferEvent>

const logs = await client.filter.getLogs({ filter })
// @log: [
// @log:   {
// @log:     address: '0xfba3912ca04dd458c843e2ee08967fc04f3579c2',
// @log:     args: { from: '0xd8da...', to: '0xa5cc...', value: 420n },
// @log:     blockHash: '0xabe6...01a4',
// @log:     blockNumber: 19760236n,
// @log:     eventName: 'Transfer',
// @log:     logIndex: 271,
// @log:     transactionHash: '0xcfa5...6e93',
// @log:     transactionIndex: 145,
// @log:   },
// @log: ]
```

## Recipes

These recipes call the action standalone with a Client as the first argument. Each is equivalent to the decorator form (`client.filter.getLogs(...)`).

### Decoding Logs

Pass `abiEvent` on the filter to decode the returned logs.

```ts twoslash
import { AbiEvent } from 'ox'
import { Actions, Client, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({ chain: mainnet, transport: http() })
// ---cut---
const transferEvent = AbiEvent.from(
  'event Transfer(address indexed from, address indexed to, uint256 value)',
)
// TODO: Replace with Actions.event.createFilter.
const id = await client.request({
  method: 'eth_newFilter',
  params: [{ topics: AbiEvent.encode(transferEvent).topics }],
})
const filter = {
  abiEvent: transferEvent, // [!code focus]
  id,
  request: client.request,
  type: 'event',
} satisfies Actions.filter.Filter<'event', typeof transferEvent>

const logs = await Actions.filter.getLogs(client, { filter })
//    ^?
```

### Filtering by Indexed Arguments

Pass `args` on the filter to filter decoded logs by indexed arguments.

```ts twoslash
import { AbiEvent } from 'ox'
import { Actions, Client, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({ chain: mainnet, transport: http() })
// ---cut---
const transferEvent = AbiEvent.from(
  'event Transfer(address indexed from, address indexed to, uint256 value)',
)
// TODO: Replace with Actions.event.createFilter.
const id = await client.request({
  method: 'eth_newFilter',
  params: [{ topics: AbiEvent.encode(transferEvent).topics }],
})
const filter = {
  abiEvent: transferEvent,
  args: { // [!code focus]
    from: '0xd8da6bf26964af9d7eed9e03e53415d37aa96045', // [!code focus]
  }, // [!code focus]
  id,
  request: client.request,
  type: 'event',
} satisfies Actions.filter.Filter<'event', typeof transferEvent>

const logs = await Actions.filter.getLogs(client, { filter })
```

## Return Value

`readonly Log[]`

The logs held by the event filter, decoded by `filter.abiEvent` when provided.

## Parameters

### filter

- **Type:** `Actions.filter.Filter<'event'>`

The event filter to fetch logs for. Its `request` function should point at the transport that created the filter id.

```ts twoslash
import { AbiEvent } from 'ox'
import { Actions, Client, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({ chain: mainnet, transport: http() })
const transferEvent = AbiEvent.from(
  'event Transfer(address indexed from, address indexed to, uint256 value)',
)
// ---cut---
// TODO: Replace with Actions.event.createFilter.
const id = await client.request({
  method: 'eth_newFilter',
  params: [{ topics: AbiEvent.encode(transferEvent).topics }],
})
const filter = { // [!code focus]
  abiEvent: transferEvent, // [!code focus]
  id, // [!code focus]
  request: client.request, // [!code focus]
  type: 'event', // [!code focus]
} satisfies Actions.filter.Filter<'event', typeof transferEvent> // [!code focus]

const logs = await Actions.filter.getLogs(client, { filter })
```
