# Public Actions

## Overview

**Public Actions** map one-to-one with "public" Ethereum RPC methods (`eth_call`, `eth_getBalance`, `eth_getBlockByNumber`, etc.). They read state and simulate calls against a node, decoding the JSON-RPC response into ergonomic, typed values.

Attach [`publicActions`](#publicactions) to a Client to call them as methods, or call each action standalone by passing a Client as the first argument from the `Actions` namespace:

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const block = await client.getBlock()
```

```ts twoslash [Standalone]
import { Actions } from 'viem'
import { client } from './viem.config'

const block = await Actions.getBlock(client)
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client, http, publicActions } from 'viem'
import { mainnet } from 'viem/chains'

export const client = Client.create({
  chain: mainnet,
  transport: http(),
}).extend(publicActions())
```

:::

## publicActions

Returns a bag of public actions bound to a Client. Pass it to [`.extend()`](/docs/clients/create#extending-a-client).

### Usage

```ts twoslash
import { Client, http, publicActions } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http(),
}).extend(publicActions())

const block = await client.getBlock()
```

### Return Value

`(client: Client) => Decorator`

A function that decorates a Client with every public action as a method.
