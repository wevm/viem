# sendRawSync

Sends a **signed** serialized transaction to the network synchronously (`eth_sendRawTransactionSync`), and returns the transaction receipt.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const receipt = await client.transaction.sendRawSync({
  transaction: '0x02f850018203118080825208808080c080a04012522854168b27e5dc3d5839bab5e6b39e1a0ffd343901ce1622e3d64b48f1a04e00902ae0502c4728cbf12156290df99c3ed7de85b1dbfe20b5c36931733a33',
})
// @log: {
// @log:   status: 'success',
// @log:   transactionHash: '0x...',
// @log: }
```

```ts twoslash [Standalone]
import { Actions } from 'viem'
import { client } from './viem.config'

const receipt = await Actions.transaction.sendRawSync(client, {
  transaction: '0x02f850018203118080825208808080c080a04012522854168b27e5dc3d5839bab5e6b39e1a0ffd343901ce1622e3d64b48f1a04e00902ae0502c4728cbf12156290df99c3ed7de85b1dbfe20b5c36931733a33',
})
// @log: {
// @log:   status: 'success',
// @log:   transactionHash: '0x...',
// @log: }
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client, http, walletActions } from 'viem'
import { mainnet } from 'viem/chains'

export const client = Client.create({
  chain: mainnet,
  transport: http(),
}).extend(walletActions())
```

:::

## Return Value

`TransactionReceipt`

The transaction receipt.

## Parameters

### requestOptions

- **Type:** `{ dedupe?: boolean; signal?: AbortSignal; ... }`

Per-request transport options forwarded to the `eth_sendRawTransactionSync` request.

### throwOnReceiptRevert

- **Type:** `boolean`

Whether to throw if the returned receipt status is `reverted`.

### timeout

- **Type:** `number`

Timeout, in milliseconds, for the sync RPC response.

### transaction

- **Type:** `Hex`

The signed serialized transaction.

## Errors

| Error | Description |
| --- | --- |
| `RpcResponse.BaseError` | The node rejected the transaction. |
| `TransactionReceipt.RevertedError` | The transaction receipt status was `reverted` and `throwOnReceiptRevert` was enabled. |
