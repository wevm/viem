# Wallet Actions

## Overview

**Wallet Actions** create, sign, and broadcast transactions. They require an account, and use either a Local Account for local signing or a JSON-RPC Account for wallet/node signing.

Attach [`walletActions`](#walletactions) to a Client to call them as methods, or call each action standalone by passing a Client as the first argument from the `Actions` namespace:

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

const hash = await client.transaction.send({
  to: '0x70997970c51812dc3a010c7d01b50e0d17dc79c8',
  value: 1n,
})
```

```ts twoslash [Standalone]
import { Actions } from 'viem'
import { client } from './viem.config'

const hash = await Actions.transaction.send(client, {
  to: '0x70997970c51812dc3a010c7d01b50e0d17dc79c8',
  value: 1n,
})
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Account, Client, http, walletActions } from 'viem'
import { mainnet } from 'viem/chains'

export const client = Client.create({
  account: Account.fromPrivateKey('0x...'),
  chain: mainnet,
  transport: http(),
}).extend(walletActions())
```

:::

## walletActions

Returns a bag of wallet actions bound to a Client. Pass it to [`.extend()`](/docs/clients/create#extending-a-client).

### Usage

```ts twoslash
import { Account, Client, http, walletActions } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  account: Account.fromPrivateKey('0x...'),
  chain: mainnet,
  transport: http(),
}).extend(walletActions())

const hash = await client.transaction.send({
  to: '0x70997970c51812dc3a010c7d01b50e0d17dc79c8',
  value: 1n,
})
```

### Return Value

`(client: Client) => Decorator`

A function that decorates a Client with every wallet action as a method.
