# waitForCallsStatus

Waits for the status & receipts of a call batch that was sent via [`sendCalls`](/docs/actions/wallet/sendCalls) ([EIP-5792](https://eips.ethereum.org/EIPS/eip-5792)).

## Usage

This example waits for a call batch's status and receipts.

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

const { receipts, status } = await client.wallet.waitForCallsStatus({
  id: '0xdeadbeef',
})
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client, custom, walletActions } from 'viem'

declare const provider: { request(...args: any): Promise<any> }

export const client = Client.create({
  transport: custom(provider),
}).extend(walletActions())
```

:::

### Standalone Action

Call `Actions.wallet.waitForCallsStatus` directly by passing the Client as the first argument.

:::code-group

```ts twoslash [example.ts]
import { Actions } from 'viem'
import { client } from './viem.config'

const { receipts, status } = await Actions.wallet.waitForCallsStatus(client, {
  id: '0xdeadbeef',
})
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client, custom } from 'viem'

declare const provider: { request(...args: any): Promise<any> }

export const client = Client.create({
  transport: custom(provider),
})
```

:::

## Return Value

The status & receipts of the call bundle. See [`getCallsStatus`](/docs/actions/wallet/getCallsStatus#return-value).

## Parameters

### id

- **Type:** `string`

The identifier of the call batch to wait for.

```ts twoslash
import { Actions } from 'viem'
import { client } from './viem.config'
// ---cut---
const { receipts, status } = await Actions.wallet.waitForCallsStatus(client, {
  id: '0xdeadbeef', // [!code focus]
})
```

### pollingInterval

- **Type:** `number`
- **Default:** `client.pollingInterval`

Polling frequency (in ms).

```ts twoslash
import { Actions } from 'viem'
import { client } from './viem.config'
// ---cut---
const { receipts, status } = await Actions.wallet.waitForCallsStatus(client, {
  id: '0xdeadbeef',
  pollingInterval: 1_000, // [!code focus]
})
```

### retryCount

- **Type:** `number`
- **Default:** `4`

Number of times to retry if the call bundle failed.

```ts twoslash
import { Actions } from 'viem'
import { client } from './viem.config'
// ---cut---
const { receipts, status } = await Actions.wallet.waitForCallsStatus(client, {
  id: '0xdeadbeef',
  retryCount: 4, // [!code focus]
})
```

### retryDelay

- **Type:** `number | (({ count }) => number)`
- **Default:** exponential backoff

Time to wait (in ms) between retries.

```ts twoslash
import { Actions } from 'viem'
import { client } from './viem.config'
// ---cut---
const { receipts, status } = await Actions.wallet.waitForCallsStatus(client, {
  id: '0xdeadbeef',
  retryDelay: ({ count }) => count * 200, // [!code focus]
})
```

### status

- **Type:** `(status) => boolean`
- **Default:** `(status) => status >= 200`

Predicate that determines which status to stop waiting on.

```ts twoslash
import { Actions } from 'viem'
import { client } from './viem.config'
// ---cut---
const { receipts, status } = await Actions.wallet.waitForCallsStatus(client, {
  id: '0xdeadbeef',
  status: ({ statusCode }) => statusCode >= 200, // [!code focus]
})
```

### throwOnFailure

- **Type:** `boolean`
- **Default:** `false`

Whether to throw an error if the call bundle fails.

```ts twoslash
import { Actions } from 'viem'
import { client } from './viem.config'
// ---cut---
const { receipts, status } = await Actions.wallet.waitForCallsStatus(client, {
  id: '0xdeadbeef',
  throwOnFailure: true, // [!code focus]
})
```

### timeout

- **Type:** `number`
- **Default:** `60_000`

Timeout (in ms) to wait before stopping polling.

```ts twoslash
import { Actions } from 'viem'
import { client } from './viem.config'
// ---cut---
const { receipts, status } = await Actions.wallet.waitForCallsStatus(client, {
  id: '0xdeadbeef',
  timeout: 30_000, // [!code focus]
})
```
