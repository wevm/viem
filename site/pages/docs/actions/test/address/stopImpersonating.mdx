# stopImpersonating

Stops impersonating an account after having previously used `impersonate`.

## Usage

This example stops impersonating an account after having previously used `impersonate`.

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

await client.address.stopImpersonating({
  address: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
})
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client, http, testActions } from 'viem'
import { anvil } from 'viem/chains'

export const client = Client.create({
  chain: anvil,
  transport: http(),
}).extend(testActions())
```

:::

### Standalone Action

Call `Actions.address.stopImpersonating` directly by passing the Client as the first argument.

:::code-group

```ts twoslash [example.ts]
import { Actions } from 'viem'
import { client } from './viem.config'

await Actions.address.stopImpersonating(client, {
  address: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
})
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client, http } from 'viem'
import { anvil } from 'viem/chains'

export const client = Client.create({
  chain: anvil,
  transport: http(),
})
```

:::

## Parameters

### address

- **Type:** `Address`

The account to stop impersonating.

```ts twoslash
import { Actions, Client, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http(),
})
// ---cut---
await Actions.address.stopImpersonating(client, {
  address: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266', // [!code focus]
  mode: 'hardhat',
})
```

### mode

- **Type:** `'anvil' | 'hardhat' | 'ganache'`
- **Default:** `'anvil'`

The test node implementation to target.

```ts twoslash
import { Actions, Client, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http(),
})
// ---cut---
await Actions.address.stopImpersonating(client, {
  address: '0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266',
  mode: 'hardhat', // [!code focus]
})
```
