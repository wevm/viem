# setIntervalMining

Sets the automatic mining interval (in seconds) of blocks. Setting the interval to `0` disables automatic mining.

## Usage

This example sets the automatic mining interval (in seconds) of blocks.

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

await client.block.setIntervalMining({
  interval: 5,
})
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client, http, testActions } from 'viem'
import { anvil } from 'viem/chains'

export const client = Client.create({
  chain: anvil,
  transport: http(),
}).extend(testActions())
```

:::

### Standalone Action

Call `Actions.block.setIntervalMining` directly by passing the Client as the first argument.

:::code-group

```ts twoslash [example.ts]
import { Actions } from 'viem'
import { client } from './viem.config'

await Actions.block.setIntervalMining(client, {
  interval: 5,
})
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client, http } from 'viem'
import { anvil } from 'viem/chains'

export const client = Client.create({
  chain: anvil,
  transport: http(),
})
```

:::

## Parameters

### interval

- **Type:** `number`

The mining interval (in seconds).

```ts twoslash
import { Actions, Client, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http(),
})
// ---cut---
await Actions.block.setIntervalMining(client, {
  interval: 5, // [!code focus]
  mode: 'hardhat',
})
```

### mode

- **Type:** `'anvil' | 'hardhat' | 'ganache'`
- **Default:** `'anvil'`

The test node implementation to target.

```ts twoslash
import { Actions, Client, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http(),
})
// ---cut---
await Actions.block.setIntervalMining(client, {
  interval: 5,
  mode: 'hardhat', // [!code focus]
})
```
