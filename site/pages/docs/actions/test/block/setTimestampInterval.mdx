# setTimestampInterval

Sets a block timestamp `interval`. The timestamp of future blocks will be computed as `lastBlock_timestamp + interval`.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

await client.block.setTimestampInterval({
  interval: 5,
})
```

```ts twoslash [Standalone]
import { Actions } from 'viem'
import { client } from './viem.config'

await Actions.test.block.setTimestampInterval(client, {
  interval: 5,
})
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client, http, testActions } from 'viem'
import { anvil } from 'viem/chains'

export const client = Client.create({
  chain: anvil,
  transport: http(),
}).extend(testActions())
```

:::

## Parameters

### interval

- **Type:** `number`

The interval (in seconds) between block timestamps.

```ts twoslash
import { Actions, Client, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http(),
})
// ---cut---
await Actions.test.block.setTimestampInterval(client, {
  interval: 5, // [!code focus]
  mode: 'hardhat',
})
```

### mode

- **Type:** `'anvil' | 'hardhat' | 'ganache'`
- **Default:** `'anvil'`

The test node implementation to target.

```ts twoslash
import { Actions, Client, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http(),
})
// ---cut---
await Actions.test.block.setTimestampInterval(client, {
  interval: 5,
  mode: 'hardhat', // [!code focus]
})
```
