# setLoggingEnabled

Enables or disables logging on the test node network.

## Usage

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

await client.node.setLoggingEnabled({
  enabled: true,
})
```

```ts twoslash [Standalone]
import { Actions } from 'viem'
import { client } from './viem.config'

await Actions.node.setLoggingEnabled(client, {
  enabled: true,
})
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client, http, testActions } from 'viem'
import { anvil } from 'viem/chains'

export const client = Client.create({
  chain: anvil,
  transport: http(),
}).extend(testActions())
```

:::

## Parameters

### enabled

- **Type:** `boolean`

Whether to enable or disable logging.

```ts twoslash
import { Actions, Client, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http(),
})
// ---cut---
await Actions.node.setLoggingEnabled(client, {
  enabled: true, // [!code focus]
  mode: 'hardhat',
})
```

### mode

- **Type:** `'anvil' | 'hardhat' | 'ganache'`
- **Default:** `'anvil'`

The test node implementation to target.

```ts twoslash
import { Actions, Client, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http(),
})
// ---cut---
await Actions.node.setLoggingEnabled(client, {
  enabled: true,
  mode: 'hardhat', // [!code focus]
})
```
