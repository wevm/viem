# Actions

## Overview

**Actions** are functions that perform JSON-RPC requests against a [Client](/docs/clients). Each action takes a Client as its first argument, and is also available as a method once the matching decorator is attached to the Client via [`.extend()`](/docs/clients/create#extending-a-client).

Decorators (such as [`testActions`](/docs/actions/test)) attach a group of actions to a Client as methods. Import them from `viem`, or call each action standalone by passing a Client as the first argument from the `Actions` namespace:

:::code-group

```ts twoslash [Decorator]
import { client } from './viem.config'

await client.block.mine({ blocks: 1 })
```

```ts twoslash [Standalone]
import { Actions } from 'viem'
import { client } from './viem.config'

await Actions.test.block.mine(client, { blocks: 1 })
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
import { Client, http, testActions } from 'viem'
import { anvil } from 'viem/chains'

export const client = Client.create({
  chain: anvil,
  transport: http(),
}).extend(testActions())
```

:::
