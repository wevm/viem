# Token Actions

Viem provides type-safe, chain-aware Actions for interacting with ERC-20 tokens. The
`tokenActions()` decorator attaches a single `token` namespace to the Client.

Each Action selects its token by `token` name (resolved from the
[chain's `tokens`](/docs/chains/introduction)), so actions like 
[`transfer`](/docs/token/transfer) and [`getBalance`](/docs/token/getBalance) need no lookups.

## Usage

The `tokenActions` decorator is available from the `viem` entrypoint.

```ts twoslash
import { createClient, tokenActions, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = createClient({
  chain: mainnet,
  transport: http(),
}).extend(tokenActions())

const balance = await client.token.getBalance({
  account: '0x55FE002aefF02F77364de339a1292923A15844B8',
  token: 'usdc',
})
// @log: {
// @log:   amount: 1234560000n,
// @log:   decimals: 6,
// @log:   formatted: '1234.56',
// @log: }
```

:::info
`token: 'usdc'` is resolved from `mainnet.tokens.usdc`, so its address
(`0xA0b8…48`) and decimals (`6`) are resolved from the chain.
:::

## Recipes

### Selecting a Token

Every Action selects its token via the `token` parameter, which is either:

- **A token name** — declared on the connected chain's `tokens` config, resolved to its contract
  `address` and `decimals`.
- **A contract `address`** — for any token, declared or not. `decimals` is inferred from the chain
  when the address matches a declared token, and otherwise defaults to `0` (used to parse `amount`
  for writes and to format the returned amount for reads). Pass `decimals` to override.

```ts twoslash
import { createClient, tokenActions, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = createClient({ chain: mainnet, transport: http() }).extend(tokenActions())
// ---cut---
// By token name (resolved from the chain).
await client.token.getBalance({ account: '0x…', token: 'usdc' })
// @log: { amount: 1000000000n, decimals: 6, formatted: '1000' }

// By contract address (undeclared, so `decimals` defaults to 0).
await client.token.getBalance({
  account: '0x…',
  token: '0x6B175474E89094C44Da98b954EedeAC495271d0F'
})
// @log: { amount: 1000000000000000000000n, decimals: 0, formatted: '1000000000000000000000' }
```

### Declaring Tokens on a Chain

Tokens are declared under the chain's `tokens` config. Every token there becomes selectable by name
from `client.token`, with no per-token code. Each token declares its contract `address`, `decimals`,
and `type` (with optional `name` and `symbol`). Built-in viem chains ship a `usdc` token where
available.

For a custom chain, pass `tokens` to [`defineChain`](/docs/chains/introduction):

```ts twoslash
import { createClient, defineChain, tokenActions, http } from 'viem'
import { privateKeyToAccount } from 'viem/accounts'

const chain = defineChain({
  id: 1,
  name: 'Ethereum',
  nativeCurrency: { name: 'Ether', symbol: 'ETH', decimals: 18 },
  rpcUrls: { default: { http: ['https://eth.merkle.io'] } },
  tokens: {
    usdc: { address: '0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48', decimals: 6, type: 'erc20' },
    dai: { address: '0x6B175474E89094C44Da98b954EedeAC495271d0F', decimals: 18, type: 'erc20' },
  },
})

const client = createClient({
  account: privateKeyToAccount('0x...'),
  chain,
  transport: http(),
}).extend(tokenActions())

// both `token: 'usdc'` and `token: 'dai'` are inferred from the chain
await client.token.transfer({ amount: '10.5', to: '0x…', token: 'dai' })
```

To extend an existing viem chain with an extra token, spread its `tokens`:

```ts twoslash
import { mainnet } from 'viem/chains'
// ---cut---
const chain = {
  ...mainnet,
  tokens: {
    ...mainnet.tokens,
    dai: { address: '0x6B175474E89094C44Da98b954EedeAC495271d0F', decimals: 18, type: 'erc20' },
  },
} as const
```

### Standalone Actions

The `token` Actions are also available standalone from `viem/actions`, taking the `client` as the
first argument. They accept the same token selector (`token` name or contract address).

```ts twoslash
import { createClient, http } from 'viem'
import { mainnet } from 'viem/chains'
import { token } from 'viem/actions'

const client = createClient({ chain: mainnet, transport: http() })

const balance = await token.getBalance(client, {
  account: '0x55FE002aefF02F77364de339a1292923A15844B8',
  token: 'usdc',
})
```
