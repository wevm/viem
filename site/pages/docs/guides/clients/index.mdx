# Clients & Transports

## Overview

The [Client](/docs/clients) is the composition root of Viem. It binds a [Chain](/docs/chains), a
[Transport](/docs/transports), an optional [Account](/docs/accounts), and opts into behavior
through [`.extend()`](/docs/clients/create#extending-a-client). These guides cover setting a
Client up, resolving Clients across several chains, and composing Transports that stay reliable
through provider failures, rate limits, and long-lived subscriptions.

```ts twoslash
import { Client, http, publicActions } from 'viem'
import { mainnet } from 'viem/chains'

const client = Client.create({
  chain: mainnet,
  transport: http(),
}).extend(publicActions())

const blockNumber = await client.block.getNumber()
// @log: 19868020n
```
