# Authorizations

## Overview

EIP-7702 lets an externally owned account authorize contract code to execute on its behalf. Viem
can prepare and sign an authorization, include it in a transaction, inspect the resulting
delegation, and call an ERC-7821 execution interface exposed by the delegated account.
