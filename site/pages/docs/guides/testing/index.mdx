# Testing

## Overview

[`testActions`](/docs/actions/test) extends a Client with methods for Anvil, Hardhat, or Ganache.
Tests can fork live networks, impersonate Accounts, manipulate balances and storage, control mining
and time, snapshot state, and inspect the transaction pool.
