### blockNumber (optional)

- **Type:** `bigint`

Block number to read the state from.

### blockOverrides (optional)

- **Type:** `BlockOverrides`

Block overrides to apply to the state.

### blockTag (optional)

- **Type:** `BlockTag`

Block tag to read the state from.

### stateOverride (optional)

- **Type:** `StateOverride`

State override to apply.
